# 問題名: ログ集計ミニツール

## 問題概要
アクセスログ風の文字列を複数行入力として受け取り、各ステータスコードが何回出現したかを集計してください。

## 入力
標準入力から複数行受け取る。  
各行は次の形式とする。

IPアドレス ステータスコード

ただし、最後に `END` が1行だけ入力されたら終了する。

### 入力例
192.168.0.1 200
10.0.0.5 404
192.168.0.1 200
172.16.1.3 500
10.0.0.5 404
END

## 出力
ステータスコードごとの出現回数を、ステータスコードの昇順で次の形式で出力する。

ステータスコード 回数

### 出力例
200 2
404 2
500 1

## 制約
- Pythonの標準ライブラリのみ使用
- 入力件数は1件以上
- ステータスコードは整数

## ヒント
- `dict` で回数を数える
- `split()` で1行を分割する
- 最後にキーを昇順ソートして表示する
