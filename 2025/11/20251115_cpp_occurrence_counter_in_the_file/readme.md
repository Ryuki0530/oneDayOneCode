# ファイル中の出現回数カウンタ（C++）

標準入力から **ファイル名** を受け取り、そのファイルを読み込み、
ファイル内に登場する **単語の出現回数** を数えるプログラムを作成せよ。

## 要件

- 単語は「英数字の連続」を 1 語とみなすこと。
- 大文字 / 小文字は区別しないものとする。
- 結果は **出現回数が多い順** に並べて出力すること。
- 同数の場合は **辞書順（昇順）** とする。
- ファイルが開けなかった場合はエラーメッセージを表示して終了する。

## 入出力例

```
input:
data.txt

output:
the 15
and 11
cat 2
hello 1
```

## 注意
- STL の `map`, `unordered_map`, `vector`, `regex`, `algorithm` など自由に使ってよい。
- 入力ファイルは UTF-8 を想定する。
- 行単位で読み込んでよい。
