# 今日の課題（Java）: シンプルTTLカウンタ

## 問題概要
時刻付きイベントを処理し、直近W秒以内のイベント数を管理するプログラムを作成してください。

## 入力形式

1行目：
Q W

- Q: クエリ数（1 ≤ Q ≤ 10^5）
- W: 時間ウィンドウ（秒）

続くQ行：
ADD t
COUNT t

- ADD t : 時刻 t にイベントを追加
- COUNT t : 時刻 t において、t-W < event_time ≤ t を満たすイベント数を出力

## 出力形式

COUNT クエリごとに1行出力

## 入力例

7 5
ADD 1
ADD 2
COUNT 2
ADD 7
COUNT 7
ADD 8
COUNT 8

## 出力例

2
1
2

## 制約・ポイント

- クエリは時刻順に与えられる
- 高速に処理する必要あり（O(Q)〜O(Q log Q)）
- キュー（Deque）を使うと良い

## ヒント

- 古いイベントは削除していく
- while (先頭 <= t - W) で取り除く

## 発展（余裕があれば）

- PEAK クエリを追加して最大イベント数を出す
- ユーザ別にカウントするよう拡張
