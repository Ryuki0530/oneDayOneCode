# Today's Problem: Consecutive Value Counter

## 問題文
整数が空白区切りで 1 行に入力されます。  
この列の中で、**同じ値が連続している区間の長さ**を順に出力してください。

ただし、出力は `値:回数` の形式で、各結果を改行して表示してください。

## 入力例
```text
1 1 1 2 2 5 5 5 5 3
```

## 出力例
```text
1:3
2:2
5:4
3:1
```

## 仕様
- 入力は 1 行
- 整数は 1 個以上含まれる
- 個数は未定なので、行全体を読み取って処理すること
- 同じ整数が連続している部分ごとに集計すること

## ヒント
- `fgets` で 1 行読み込む
- `strtok` で空白区切りに分割する
- 前の値と現在の値を比較しながら数える

## 目標
配列をただ数えるのではなく、**連続区間ごとの処理**に慣れること。

## 追加課題
余裕があれば、入力が

```text
7
```

のように 1 個だけのときも正しく動くようにしてください。
```
