### 🧮 **問題：数値の桁和を求めよ**

#### 【問題文】
整数 \( N \) が与えられる。  
この数の **各桁の合計（桁和）** を出力せよ。

#### 【入力形式】
```
N
```

#### 【出力形式】
```
sum
```

#### 【入力例1】
```
12345
```

#### 【出力例1】
```
15
```

#### 【入力例2】
```
9009
```

#### 【出力例2】
```
18
```

---

#### 【ヒント】
- 数字を文字列として扱うと簡単です。  
  例：`sum(int(c) for c in str(N))`
