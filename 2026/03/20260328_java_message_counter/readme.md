# 今日の課題（Java）

## 問題名
Recent Unique Messages

## 問題文
チャットのメッセージログが時系列順に与えられます。
各行には `ユーザー名` と `メッセージ内容` が含まれます。

ただし、**同じユーザーが直前に送ったメッセージと完全に同じ内容**を続けて送った場合、そのメッセージは重複として無視します。

重複を除いたあと、各ユーザーが送信した有効メッセージ数を数え、
**件数の降順、ユーザー名の辞書順昇順**で出力してください。

## 入力
- 1行目に整数 `N`
- 続く `N` 行に `user message`

## 制約
- `1 <= N <= 1000`
- `user` は英数字のみ
- `message` は空白を含まない文字列

## 判定ルール
あるユーザーについて、
- そのユーザーの直前のメッセージと今回のメッセージが同じなら無視
- そうでなければ有効

※ 他ユーザーの発言は関係ありません。

## 出力
有効メッセージ数を集計し、以下の形式で出力してください。

`user count`

## 入力例
```
8
alice hello
bob hi
alice hello
alice hello
bob hi
bob ok
alice bye
bob ok
```
## 出力例
```
alice 2
bob 2
```
## 説明
- `alice hello`（1回目）: 有効
- `bob hi`（1回目）: 有効
- `alice hello`（2回目）: alice の直前メッセージと同じなので無視
- `alice hello`（3回目）: 直前も同じなので無視
- `bob hi`（2回目）: bob の直前メッセージと同じなので無視
- `bob ok`: 有効
- `alice bye`: 有効
- `bob ok`（2回目）: bob の直前メッセージと同じなので無視

よって、
- alice: 2件
- bob: 2件

## ヒント
- ユーザーごとに「直前のメッセージ」を記録する
- ユーザーごとの有効件数を `Map` で管理する
- 最後に一覧をソートして出力する

## 目標
- `HashMap`
- 文字列比較
- `List` への詰め替えと `sort`
- 入力処理の基本

## 余裕があれば
同点のときの並び順を
- 辞書順昇順
- 辞書順降順
で切り替えられるようにしてみてください。
