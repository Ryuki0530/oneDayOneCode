# 📄 今日のミニ課題（Python / OpenCV）
**テーマ：リアルタイム・ドキュメントスキャナ（最小機能版）**

## 要件
- 第1引数に**カメラID（整数）**を受け取る（例：`0`）。
- 起動中、**台形（紙）を検出している間だけ** `SPACE` キーで**撮影**可能。
- 撮影画像は**透視変換後**（正面化）の1枚を**カレントディレクトリ**へ自動保存（`scan_YYYYmmdd_HHMMSS.png`）。
- `q` / `ESC` で終了。
- 依存は `opencv-python`, `numpy` のみ。

## 使い方（例）
```bash
python scan_cam.py 0
