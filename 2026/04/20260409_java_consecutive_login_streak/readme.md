# Daily Coding Challenge - Consecutive Login Streak

## 問題概要
ユーザーのログイン結果が日ごとに記録されています。  
連続して `1` が続いた最長日数を求めてください。

## 問題
1行目に整数 `N` が与えられます。  
2行目に `N` 個の整数 `0` または `1` が空白区切りで与えられます。

- `1` はログインした日
- `0` はログインしなかった日

このとき、**連続してログインした最長日数**を出力してください。

## 入力形式
- 1行目: 整数 `N`
- 2行目: `0` または `1` を `N` 個、空白区切りで入力

## 出力形式
- 連続して `1` が続いた最大個数を出力

## 制約
- `1 <= N <= 1000`

## 入力例1
```text
7
1 1 0 1 1 1 0
```

## 出力例1
```text
3
```

## 説明
`1 1` の連続は2日、  
`1 1 1` の連続は3日なので、答えは `3` です。

## ヒント
- 今の連続数を数える変数
- 最大値を保持する変数

この2つを使うと簡単に解けます。
```
