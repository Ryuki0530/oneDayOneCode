# 🧠 C言語課題：数字の階段パターン出力

## 問題文

標準入力から正の整数 `N` を受け取り、以下のような「数字の階段」を表示するプログラムを作成してください。

ただし、各段の数字はその段の番号（1始まり）を表示してください。

---

### 🔸 入力

整数 `N`（1 ≦ N ≦ 20）

---

### 🔹 出力

段数 `N` の数字階段を出力する。  
各行にはその行番号に対応する数字を、行番号回繰り返して表示する。

---

### 🔸 入力例
```
4
```
---

### 🔹 出力例
```
1
22
333
4444d
```
---

## ✏️ 補足

- `printf()` を使って出力してください。
- ループ処理（`for`）を使いましょう。
- バッファサイズを考慮する必要はありません。
