# 今日の課題: Motor Command Safety Filter

## 概要

ロボットのモータに送る速度指令値を、安全範囲内に補正するプログラムを作成してください。

各時刻で与えられる速度指令値 v に対して、以下のルールで補正します。

- v < -100 の場合、-100 に補正する
- v > 100 の場合、100 に補正する
- それ以外の場合、そのまま使う

さらに、補正が発生した回数を数えてください。

## 入力

N
v1 v2 v3 ... vN

- N は指令値の個数
- vi は整数の速度指令値

## 出力

補正後の指令値を1行で出力してください。
その次の行に、補正が発生した回数を出力してください。

## 制約

1 <= N <= 100
-1000 <= vi <= 1000

## 入力例

6
120 80 -150 0 100 -100

## 出力例

100 80 -100 0 100 -100
2

## 実装条件

- C++で実装すること
- vector<int> を使うこと
- 補正処理は関数 clampCommand として分けること

関数例:

int clampCommand(int v);

## 発展

余裕があれば、補正前と補正後が異なる場合だけ、補正回数をカウントしてください。
