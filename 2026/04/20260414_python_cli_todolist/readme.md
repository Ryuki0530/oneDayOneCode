# 今日の課題: CLI TODOリスト

## 問題名
CLI TODO List

## 概要
CSVファイルを用いてタスクを保存・管理する、CLI形式のTODOリストツールを作成してください。

このツールでは、タスクの追加、一覧表示、完了処理を行えるものとします。

---

## 使用言語
Python

---

## 保存形式
タスクはCSVファイルに保存してください。

1行につき1タスクを保存し、以下の情報を持つものとします。

- タスクID
- 完了状態
- タスク名
- 期限
- 補足メッセージ

### データ例
- タスクID: 1, 2, 3, ...
- 完了状態: `0` = 未完了, `1` = 完了
- タスク名: 任意の文字列
- 期限: `yyyymmddhhmm` 形式の日時文字列
- 補足メッセージ: 任意の文字列（空でも可）

---

## CLI仕様

### 1. オプションなしで実行
コマンド:

    python todo.py

全タスクについて、以下を表示してください。

- タスクID
- タスク名
- 期限
- 補足メッセージ（存在する場合のみ）

表示形式は自由ですが、見やすく整形してください。

---

### 2. タスク追加
コマンド:

    python todo.py add タスク名 期限

例:

    python todo.py add レポート提出 202604201700

指定したタスク名・期限で、新しいタスクを追加してください。

追加時の初期状態は未完了とします。

---

### 3. 補足メッセージ付きで追加
コマンド:

    python todo.py add タスク名 期限 -m 補足メッセージ

例:

    python todo.py add レポート提出 202604201700 -m 情報数学の課題

補足メッセージ付きでタスクを追加してください。

---

### 4. 毎週の繰り返し追加
コマンド:

    python todo.py add タスク名 期限 -r 回数

例:

    python todo.py add ゼミ 202604151400 -r 4

この場合、指定日時を基準として毎週1件ずつ、指定回数ぶんのタスクを追加してください。

上の例では、以下の4件が追加されるイメージです。

- 202604151400
- 202604221400
- 202604291400
- 202605061400

---

### 5. 毎週追加 + 補足メッセージ
コマンド:

    python todo.py add タスク名 期限 -r 回数 -m 補足メッセージ

例:

    python todo.py add ゼミ 202604151400 -r 4 -m 411教室

`-r` と `-m` は同時に使えるようにしてください。

---

### 6. タスク完了

コマンド:

    python todo.py fin タスクID

例:

    python todo.py fin 3

指定したタスクIDに一致するタスクを完了状態にしてください。

---

### 7. タスク名で完了
コマンド:

    python todo.py fin --name タスク名

例:

    python todo.py fin --name レポート提出

指定したタスク名に一致するタスクを完了状態にしてください。

一致するタスクが複数ある場合は、以下のいずれかの仕様としてください。

- 一致したものをすべて完了にする
- エラーメッセージを表示して終了する
- 該当候補を表示して選ばせる

ただし、どの仕様にしたか分かるようにしてください。
---

## 入力仕様
- タスク名は空文字であってはならない
- 期限は `yyyymmddhhmm` 形式であること
- `-r` の回数は1以上の整数であること

不正な入力があった場合は、エラーメッセージを表示してください。

---

## 実装条件
- CSVファイルを使って永続化すること
- プログラムを再実行しても前回のタスク情報が残ること
- 標準ライブラリを中心に実装すること
- CLI引数の解析方法は自由
  - `sys.argv`
  - `argparse`
  - どちらでも可

---

## 補足
期限の加算処理では、毎週追加のために「7日後」を正しく求めてください。  
文字列のまま操作するのではなく、日時として解釈してから加算する実装が望ましいです。
