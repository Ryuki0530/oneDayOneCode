# Textual を使った「じゃんけんアプリ」を作成せよ

## 概要
Python の Textual ライブラリを用いて、ターミナル内で動作する
GUI じゃんけんアプリを作成せよ。

ユーザーは **GUI のボタン（グー / チョキ / パー）** をクリックまたはキー操作で選び、
結果（勝ち・負け・あいこ）を画面内に表示する。

## 要件

### 1. 画面構成
- 画面上部：アプリタイトル  
- 中央：ユーザーの選択肢（グー・チョキ・パー）を表示するボタンを横一列に並べる  
- 下部：  
  - コンピュータの手  
  - 勝敗判定結果  
  を表示する領域を配置すること

### 2. 機能
- ユーザーがボタンを押すと、その手が確定される
- コンピュータはランダムに手を選ぶ
- 勝敗を判定し、結果欄に  
  「あなたの手：〇〇 / コンピュータ：△△ / 結果：××」  
  を表示すること
- レイアウトは Textual の `Horizontal`, `Vertical`, `Static`, `Button` などの基本ウィジェットを使うこと

### 3. 実装条件
- Textual のイベントハンドラ (`on_button_pressed`) を使用すること
- 少なくとも以下の Textual 機能を使うこと  
  - `App`  
  - `compose()`  
  - `Button`  
  - `Static`  
- Python の標準ライブラリ以外の依存は Textual のみとする

### 4. 動作確認
- `python main.py` でターミナル内に GUI が表示されること
- ボタンを押すたびに結果表示が更新されること

## 追加要件（できれば）
- キーボード操作（例：1 でグー、2 でチョキ、3 でパー）でも選択できるようにする  
- 結果の表示部分を色付けする（勝ち：緑、負け：赤、あいこ：黄色）
