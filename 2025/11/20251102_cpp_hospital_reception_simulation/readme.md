# 🌟 C++課題：病院の受付シミュレーション

## 概要
あなたは病院の受付システムを作ることになりました。  
患者は受付時に「名前」「年齢」「緊急度（1〜5）」を登録します。  
緊急度が高い（数値が大きい）患者ほど優先して診察を受けます。  
同じ緊急度の場合は、先に受付した人を優先します。

このシステムを、**優先度付きキュー（`std::priority_queue`）**を用いて実装してください。

---

## 入力
最初の行に整数 `N`（患者数）が与えられます。  
続く `N` 行には、各患者の情報が次の形式で与えられます：

名前 年齢 緊急度


名前は英数字からなる文字列、年齢と緊急度は整数です。

---

## 出力
緊急度の高い順に患者を診察順として出力してください。  
フォーマットは以下の通りです：

診察順:
<順位>. 名前(年齢) - 緊急度: <緊急度>


---

## 入出力例

### 入力例
```
5
Tanaka 40 2
Suzuki 65 5
Yamada 22 3
Ito 33 5
Kato 30 1
```
### 出力例
```
診察順:

Suzuki(65) - 緊急度: 5

Ito(33) - 緊急度: 5

Yamada(22) - 緊急度: 3

Tanaka(40) - 緊急度: 2

Kato(30) - 緊急度: 1
```
---

## ヒント
- `struct Patient { string name; int age; int urgency; int order; };`
- `order` は受付順を保持するために使います。
- 比較関数では `(urgency, order)` の順に比較します。
- `std::priority_queue<Patient, vector<Patient>, Compare>` の形で定義できます。
