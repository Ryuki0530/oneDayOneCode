# 課題：暗号文を解読せよ

## 問題文

あなたは、英単語の文章が暗号化されたテキストを受け取りました。
暗号化のルールは「アルファベットをN文字ずらすシーザー暗号」です。
N は 1〜25 の範囲で、与えられた暗号文から自動的に最も自然な英語になる N を推定し、復号文を出力してください。

## 仕様

- 入力：暗号文（例: "Uifsf jt b tfdsfu dpef."）

- 出力：復号文（例: "There is a secret code."）

推定には、以下のヒントを使ってよい：

英語の頻出単語（"the", "is", "and"など）が多いほど自然

大文字・小文字の区別は保持する

自作のスコア関数を定義して「最も自然な文」を求めること

## 目標

decode_caesar(cipher_text) 関数を定義する

出力は復号文1行

スコア関数の設計（頻出単語スコアや母音率など）を工夫すると加点

## 実行例：

- 入力:
```
cipher = "Uifsf jt b tfdsfu dpef."
print(decode_caesar(cipher))
```

- 出力:

```
There is a secret code.
```
