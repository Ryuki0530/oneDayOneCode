# 今日の課題（Java）: 最短経路つき迷路（BFSで復元）

## 問題
H×W の迷路が与えられる。各マスは以下のいずれか：
- `S`：スタート（1つ）
- `G`：ゴール（1つ）
- `.`：通行可能
- `#`：壁（通行不可）

上下左右に 1 マスずつ移動できるとき、**最短手数**で `S` から `G` へ到達せよ。

到達できるなら：
1) 最短手数（移動回数）  
2) 最短経路を `*` でマーキングした迷路（`S`,`G`はそのまま）

到達できないなら `-1` のみ出力。

## 入力
- 1行目: `H W`
- 次のH行: 迷路（長さWの文字列）

## 出力
- 到達不可: `-1`
- 到達可能:  
  1行目に最短手数  
  続くH行にマーキング後の迷路

## 制約（目安）
- 1 ≤ H, W ≤ 200

---

## 例
入力
```
3 5
S...#
##..#
...G.
```

出力
```
6
S***#
##**#
...G.
```

---

## 実装の要点（超重要）
- BFSで距離 `dist[h][w]` を埋める（最短が保証される）
- 経路復元のために `prevH[h][w], prevW[h][w]` を持つ  
  （そのマスに来た直前の座標を記録）
- `G` から `S` に向かって `prev` を辿り、途中を `*` にする

---

## 必要モジュール（使い方ミニ解説）
- `ArrayDeque<int[]>`
  - キュー（BFS）に最適。`add()`で入れて`poll()`で取り出す。
- `Arrays.fill(dist[i], -1)`
  - 未訪問を -1 初期化するのに便利。
- 入力は `BufferedReader` + `StringTokenizer` 推奨（速い）
