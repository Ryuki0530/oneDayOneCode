# C Programming Mini Task – File Word Counter

あなたの仕事は、テキストファイル中に含まれる **単語数** を数える C プログラムを作成することです。

## 要件
- 入力として **ファイル名をコマンドライン引数で受け取る**。
- 単語は「空白・改行・タブで区切られた文字列」とみなす。
- ファイルが開けなければエラーメッセージを出して終了する。
- 標準出力に以下の形式で結果を出す：

Word count: <数値>

## 入力例
./wordcount sample.txt

## 出力例
Word count: 42
