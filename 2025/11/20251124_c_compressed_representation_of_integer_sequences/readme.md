# C 課題：整数列の圧縮表現

 標準入力からスペース区切りで整数列が与えられる。

 例
 10 10 10 3 3 1 1 1 1 8

 この整数列を、同じ値が連続している区間ごとにまとめ、
 以下のように (値, 回数) の形式で出力せよ。

 10 x 3
 3 x 2
 1 x 4
 8 x 1

## 【要件】

 - 標準入力からスペース区切りの整数を読み込むこと。個数は不定とする。
 - 各整数は int 型に収まる範囲でよい。
 - 出力は上記の形式に従うこと。
 - 配列長は固定長（例えば 1000 要素）としてよい。
 - 動的メモリ確保は使用しないこと。
 - プログラムは単一の C ファイルで実装すること。

## 【補足】

 - 標準入力は scanf("%d", &x) を繰り返して読み込んでよい。
 - 最後のグループも必ず出力すること。
