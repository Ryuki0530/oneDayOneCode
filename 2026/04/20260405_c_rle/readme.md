# 今日のC課題: 連続する同一値の圧縮

## 問題
整数列が与えられます。  
左から順に見て、**同じ値が連続している部分**をひとつのまとまりとして扱い、  
`値 個数` の形式で出力してください。

## 入力
- 1行目に整数 `N`
- 2行目に整数 `N` 個

## 出力
連続する同一値ごとに、各行に以下の形式で出力してください。

`値 個数`

## 制約
- `1 <= N <= 100`
- 各整数の値は `-1000` 以上 `1000` 以下

## 入力例1
```text
8
1 1 1 2 2 5 5 1
```

## 出力例1
```text
1 3
2 2
5 2
1 1
```

## 入力例2
```text
5
7 7 7 7 7
```

## 出力例2
```text
7 5
```

## ポイント
- 「同じ値が出てきた回数」ではなく、**連続している個数**を数えます。
- 例えば `1 2 1` は、
  - `1 1`
  - `2 1`
  - `1 1`
  と出力します。

## ヒント
- 直前の値を覚える
- 値が変わったら、それまでの値と個数を出力する
- 最後のまとまりを出力し忘れないように注意

## 余力があれば
入力を1行ずつ読みながら処理して、配列を使わずに解いてみてください。
```
