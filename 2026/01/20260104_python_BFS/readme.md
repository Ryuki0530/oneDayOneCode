# 今日のPython課題：最短経路（BFS）で「最小手数」を出す

## 問題
`H×W` のグリッドが与えられる。  
`S`（スタート）から `G`（ゴール）まで、上下左右に1マスずつ移動できるときの **最小移動回数** を求めよ。  
壁 `#` は通れず、通れるマスは `.` とする。  
到達できない場合は `-1` を出力せよ。

## 入力
- 1行目：`H W`
- 続く `H` 行：長さ `W` の文字列（`S`, `G`, `.`, `#`）

## 出力
- 最小移動回数（到達不可なら `-1`）

## 制約（目安）
- `1 <= H, W <= 100`

## 例
入力：
```
3 4
S..#
.#..
...G
```

出力：
```
5
```


## 実装条件
- **BFS（幅優先探索）** を使うこと
- 距離配列（未訪問は `-1` など）を用意すること
