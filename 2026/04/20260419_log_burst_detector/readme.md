# 問題名
Log Burst Detector

## 問題
ログのイベントIDが時系列順に並んで入力されます。

同じイベントIDが**連続して 3 回以上**出現した区間を「バースト」と呼ぶことにします。  
入力された列の中から、バーストになっているイベントIDとその連続回数を、**出現順**に出力してください。

ただし、同じIDが 2 回以下しか連続していない区間は無視します。

## 入力
1 行目に整数 `N` が与えられます。  
2 行目に `N` 個の整数 `a1 a2 ... aN` が空白区切りで与えられます。

## 出力
バーストである区間ごとに、以下の形式で 1 行ずつ出力してください。

`イベントID 連続回数`

バーストが 1 つも存在しない場合は `None` と出力してください。

## 制約
- `1 <= N <= 1000`
- `0 <= ai <= 100000`

## 入力例1
```text
10
1 1 1 2 3 3 3 3 4 4
```

## 出力例1
```
1 3
3 4
```
## 入力例2
```text
8
5 5 2 2 2 7 8 8
```

## 出力例2
```
2 3
```

## 入力例3
```text
6
9 9 1 2 3 3
```

## 出力例3
```
1 3
3 4
```
