# 今日の課題（Java）
## 問題名
アクセス集計

## 問題文
標準入力から複数行のアクセスログを受け取り、ユーザーごとのアクセス回数を集計してください。

各行は次の形式です。

`ユーザー名 アクション名`

例:
`alice LOGIN`

入力の終わりまで読み込み、各ユーザーの出現回数を数えてください。

その後、ユーザー名の辞書順で、次の形式で出力してください。

`ユーザー名 回数`

## 入力例
alice LOGIN
bob VIEW
alice VIEW
charlie LOGIN
bob LOGOUT
alice LOGOUT

## 出力例
alice 3
bob 2
charlie 1

## 条件
- 1行に1件のログが与えられる
- 各行は半角スペース区切りで2項目
- 行数は1以上
- ユーザー名は英小文字のみとする
- 出力はユーザー名の辞書順

## ヒント
- `BufferedReader` で入力すると扱いやすいです
- 集計には `Map<String, Integer>` が使えます
- 辞書順で出したいので `TreeMap` も便利です

## 余裕があれば
アクション名ごとの回数も別に集計してみてください。
