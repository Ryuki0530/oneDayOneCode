# 簡易ファイル検索ツール（Java）

標準入力から以下を受け取り、該当するファイルを一覧表示するプログラムを作成せよ。

入力：
1 行目：検索対象ディレクトリのパス
2 行目：検索したい拡張子（例：txt, csv, java など）

仕様：
- 指定されたディレクトリ直下にあるファイルのうち、
  2 行目で与えられた拡張子を持つファイル名だけをすべて表示すること。
- ディレクトリは再帰的に探索しなくてよい（1 階層のみ）。
- 表示順序は自由。
- 拡張子の比較は大文字・小文字を区別しない。

例：
入力：
C:/test
txt

出力例（順不同）：
memo.txt
readme.TXT
