# Java 課題: センサー値フィルタリング

あなたは温度センサーのデータを扱う小さなツールを作成することになりました。
標準入力から整数（温度）を複数行受け取り、以下の処理を行ってください。

## 要件

1. 温度データを保持する `TemperatureRecord` クラスを作成すること。
   - フィールド: `value`（int）, `timestamp`（long, System.currentTimeMillis() を使用）
   - コンストラクタと getter を実装すること。

2. メインクラスでは、標準入力から温度値を読み込み、
   `-100` が入力されたら入力終了とする。

3. 受け取った温度データのうち、
   **20℃ 以上 30℃ 以下の値だけ**を抽出し、
   抽出した値だけを昇順で表示すること。

## 入出力例

入力:
```
15
22
31
27
-100
```

出力:
```
22
27
```
