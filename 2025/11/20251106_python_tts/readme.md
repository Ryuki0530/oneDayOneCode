# 🧩今日の課題（Python）

## 題名：音でしゃべる！Pythonナレーター

### 📜問題文
標準入力から文章を受け取り、それを音声として再生するプログラムを作成せよ。  
以下の仕様を満たすこと。

1. `pyttsx3` モジュール（音声合成ライブラリ）を使用すること。  
   ※ 外部モジュールだが非常に軽量で、オフラインで動作する。
2. 入力されたテキストを読み上げること。
3. 複数行入力に対応させること（空行で入力を終了）。
4. 読み上げが完了したら `"--- END OF SPEECH ---"` を出力すること。

---

### 💡ヒント
`pyttsx3` は OS 標準の TTS（Text-to-Speech）機能を呼び出すため、ネット接続不要で利用できる。

```bash
pip install pyttsx3
```
```python
import pyttsx3
engine = pyttsx3.init()
engine.say("Hello world")
engine.runAndWait()
```
