# 今日のJava課題：文字列の「最短周期」を求めよ（KMP応用）

## 問題
文字列 `S`（長さ `N`）が与えられる。  
`S` を **ある文字列 `T` を繰り返して**作れるとき、`T` の長さを「周期」と呼ぶ。

例：
- `S = "ababab"` は `T="ab"` を3回 → 周期は 2
- `S = "aaaa"` は `T="a"` を4回 → 周期は 1
- `S = "abac"` は繰り返しで作れない → 周期は 4（=自分自身）

**S の最短周期の長さ**を出力せよ。

## 入力
- 1行目：文字列 `S`（英小文字のみ）

## 出力
- 最短周期の長さ（整数）

## 制約
- `1 <= |S| <= 2 * 10^5`

## 例
入力：
```
ababab
```


出力：
```
2
```



入力：
```
abac
```


出力：
```
4
```



## ヒント（最小限）
- KMP の prefix function（failure function / lps）を使うと速い。
- `N - lps[N-1]` が「候補の周期長」になる。
- `N % 候補 == 0` ならそれが答え、そうでなければ `N`。

## 実装条件
- `O(N)` で解くこと（全探索の周期チェックは不可）
- `Main` クラス、標準入力/標準出力で完結
使うモジュール（超短解説）
java.io.BufferedReader：高速入力

char[]：文字アクセスを高速化

int[] lps：KMPの部分一致テーブル
