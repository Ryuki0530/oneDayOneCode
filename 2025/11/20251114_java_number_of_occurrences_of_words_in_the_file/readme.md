# 【Java課題】  
## ファイル内の単語出現回数を集計せよ（シンプル版）

標準入力で読み込んだテキストファイル（UTF-8）について、  
各単語の出現回数を数え、**出現回数の多い順に上位10件**を表示するプログラムを作成せよ。

### 要件
- Java で作成すること。
- 入力ファイル名はコマンドライン引数で受け取る。
- 「単語」は、英字または数字が連続した部分とする。  
  例：`Hello, world!` → `Hello` と `world`
- 大文字・小文字は区別しない（`Hello` と `hello` は同じ）。
- 出力形式は  
```
word count
```
の形式で上位10件を表示すること。
- ファイルが存在しない場合はエラーを表示し終了すること。

### 入力例
```
java WordCount sample.txt
```
### 出力例（形式の例）
```
the 42
to 31
and 29
```
...
