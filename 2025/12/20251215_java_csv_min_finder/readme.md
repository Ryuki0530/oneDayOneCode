# 今日の課題（Java / すぐ終わる）

## 問題：CSVの「最小限」集計ツール（標準入力→標準出力）
標準入力から `N` 行のデータを読み取り、指定された列を **合計 / 平均 / 最小 / 最大** で集計して出力してください。  
（文字数カウント系ではなく、I/Oとパース・例外処理の練習寄りです）

---

## 入力形式
1行目：`N`（データ行数）  
2行目：`col`（集計対象の列番号：0始まり）  
3行目：`mode`（集計方法。`sum` / `avg` / `min` / `max` のいずれか）  
4行目以降：CSV形式のデータ行が `N` 行  
- 各行はカンマ区切り
- 集計対象列 `col` の値は **整数** として解釈する
- ただし、次の「無効データ」はスキップする（集計に含めない）
  - 列数が足りない
  - 対象セルが空文字
  - 整数に変換できない

---

## 出力形式
- 有効データが1件以上ある場合：集計結果を1行で出力
  - `mode=sum|min|max`：整数として出力
  - `mode=avg`：小数第2位まで（例：`12.34`）で出力
- 有効データが0件の場合：`NA` を出力

---

## 制約
- `1 <= N <= 2000`
- 1行の長さは適当（常識的な範囲）
- 外部ライブラリ禁止（標準ライブラリのみ）

---

## 例

### 入力例
7
2
avg
a,b,10
x,y,20
p,q,
m,n,abc
onlyone
u,v,30
k,l,40

### 出力例
25.00

（有効な col=2 は 10,20,30,40 → 平均 25.00）

---

## 実装メモ（ヒント）
- `BufferedReader` + `String.split(",")` でOK（今回は厳密CSVでなく「簡易CSV」扱い）
- スキップ数や有効件数をちゃんと数える
- `avg` は `double` で計算して `System.out.printf("%.2f%n", value);`

---

## 追加チャレンジ（時間があれば）
- `mode` が不正な場合は `NA` を出す（例外で落とさない）
- `split(",", -1)` を使って末尾の空セルを保持してみる
