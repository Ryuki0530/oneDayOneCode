# Robot Arm Safety Gate

## 概要

ロボットアームに対して、関節角度を変更するコマンド列が与えられます。

各関節には最小角度と最大角度があり、その範囲を超える操作は安全のため拒否されます。

コマンドを順番に処理し、実行されたコマンド数、拒否されたコマンド数、最終的な各関節角度を出力してください。

## 入力

```
N Q
min_1 max_1 angle_1
min_2 max_2 angle_2
...
min_N max_N angle_N
joint_1 delta_1
joint_2 delta_2
...
joint_Q delta_Q
```

## 条件

- `N` は関節数
- `Q` はコマンド数
- `min_i` は関節 `i` の最小角度
- `max_i` は関節 `i` の最大角度
- `angle_i` は関節 `i` の初期角度
- `joint_j` は操作対象の関節番号
- `delta_j` は角度の変化量
- 関節番号は `1` から始まる
- 操作後の角度が範囲内なら実行
- 範囲外になる場合は拒否し、角度は変更しない

## 出力

```
executed rejected
angle_1 angle_2 ... angle_N
```

## 制約

- `1 <= N <= 100`
- `1 <= Q <= 1000`
- `-360 <= min_i <= angle_i <= max_i <= 360`
- `-100 <= delta_j <= 100`

## 入力例

```
3 5
-90 90 0
0 180 90
-45 45 10
1 30
2 100
3 -60
1 -50
2 -30
```

## 出力例

```
3 2
-20 60 10
```

## 説明

1. 関節1: `0 + 30 = 30` → 実行
2. 関節2: `90 + 100 = 190` → 最大値180を超えるので拒否
3. 関節3: `10 - 60 = -50` → 最小値-45を下回るので拒否
4. 関節1: `30 - 50 = -20` → 実行
5. 関節2: `90 - 30 = 60` → 実行

実行数は3、拒否数は2です。
