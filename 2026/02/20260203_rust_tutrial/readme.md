# Problem: Log Window Stats (Rust)

## 概要
標準入力で与えられるアクセスログ（時刻とユーザーID）について、
「直近K秒のユニークユーザー数」を逐次出力するプログラムを作成してください。

ログは時刻順に入力されます（同時刻が来てもよい）。

## 入力形式
1行目: N K  
2行目以降: N行のログ  
各ログは: `t user`  
- `t` は整数（秒）
- `user` は英数字の文字列（空白なし）

## 出力形式
各ログ行を処理した直後に、その時刻 `t` における
「時刻が [t-K+1, t] の範囲にあるログに登場するユニークユーザー数」を1行で出力してください。

## 制約
- 1 <= N <= 200000
- 1 <= K <= 10^9
- t は 0 以上の整数
- 入力は時刻順（非減少）
- 速度要件: O(N log N) 以内を目標（できれば O(N)〜O(N log U)）

## 例
入力:
6 3
1 alice
2 bob
2 alice
3 carol
5 bob
6 dave

出力:
1
2
2
3
1
2

## 説明（最初だけ）
K=3のとき、t=3では範囲は [1,3]。
登場ユーザーは alice, bob, carol の3人 → 3。

## 実装ヒント
- 「直近K秒」のログをキュー（VecDeque）で持つ
- ユーザーごとの出現回数を HashMap で管理
- 範囲外になったログを先頭から捨て、回数が0になったら HashMap から削除
- 出力は都度 `println!`

## 要求
- Rust 1.70+ 相当で動くコードを想定（stable）
- 標準ライブラリのみでOK
