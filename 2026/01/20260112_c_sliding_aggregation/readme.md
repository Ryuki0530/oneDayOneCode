# 今日のC課題：固定長ログの「スライディング集計」（リングバッファ）

## 問題
あなたはセンサの値（整数）を1件ずつ受け取る。直近 **W件** の合計を常に求めたい。

入力は以下の形式：
- 1行目：整数 `W`（1 ≤ W ≤ 200000）
- 2行目：整数 `N`（1 ≤ N ≤ 200000）
- 3行目以降：整数 `a_i` が `N` 行（各 `a_i` は -10^9 ～ 10^9）

出力：
- i=1..N について、直近W件（存在する範囲）の合計 `S_i` を1行ずつ出力せよ。
  - 例：W=3 のとき  
    - i=1: a1  
    - i=2: a1+a2  
    - i=3: a1+a2+a3  
    - i=4: a2+a3+a4  
    - ...

## 制約・注意
- `N` が大きいので、**O(N)** で解くこと。
- 合計は 64bit（`long long`）で持つこと（オーバーフロー対策）。

## 例
入力
3
6
10
20
30
40
-5
7

出力
10
30
60
90
65
42

## ヒント（最短）
リングバッファ `buf[W]` を用意して、
- 追加する値を `sum += x`
- すでにW件埋まっていたら、置き換えられる古い値 `old = buf[idx]` を `sum -= old`
- `buf[idx] = x`, `idx = (idx+1) % W`
- `sum` を出力

## 追加チャレンジ（余裕があれば）
直近W件の **平均（小数なし切り捨て）** も同じ行に出力せよ（合計と平均をスペース区切り）。
