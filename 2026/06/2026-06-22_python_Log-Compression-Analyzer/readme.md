# Log Compression Analyzer

## 問題

ログデータ中の連続した同一文字を圧縮するプログラムを作成せよ。

例えば、

AAABBBBBCCDAA

という文字列は

A3B5C2D1A2

と変換する。

## 要件

1. 文字列を標準入力から受け取る
2. 連続する同じ文字の個数を数える
3. 「文字 + 個数」の形式で出力する

## 実行例

入力

AAABBBBBCCDAA

出力

A3B5C2D1A2

---

入力

ABC

出力

A1B1C1

## 発展課題

圧縮後の文字列を受け取り、

A3B5C2D1A2

↓

AAABBBBBCCDAA

へ復元する関数も作成せよ。
