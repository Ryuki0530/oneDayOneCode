# Python 課題：loguru を使ったログ管理スクリプト作成

## 説明
loguru ライブラリを用いて、以下の仕様を満たすログ管理プログラムを作成せよ。

### 機能要件
1. プログラム起動時に、標準出力へ色付きログで「Program Started」を表示すること。
2. `app.log` にログを書き出すこと。
3. ログファイルはサイズが 200KB を超えたら自動的にローテーションされるよう設定すること。
4. ユーザーに数字を入力させ、その数字が偶数なら info、奇数なら warning、0 のときは error としてログに記録せよ。
5. `0` が入力された場合はプログラムを終了すること。

### 入出力例
入力： 3  
→ warning ログで「3 is odd」

入力： 10  
→ info ログで「10 is even」

入力： 0  
→ error ログで「0 received, program terminated」  
→ プログラム終了

## 注意
loguru がインストールされていない場合は `pip install loguru` で導入できる。
