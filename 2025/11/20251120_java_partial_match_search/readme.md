# 📘 今日の Java 課題（短時間版）

## 🧩 課題：部分一致検索ツールを作る

標準入力から複数行の文字列データを受け取り、  
最後に検索キーワードを1行入力すると、  
**そのキーワードを部分一致で含む行だけ出力**するプログラムを作りなさい。

### ▼ 入力形式
1. 最初に整数 `N` が与えられる（行数）
2. 続けて N 行のテキスト
3. 最後に検索キーワード

### ▼ 出力形式
検索キーワードを **含む** 行だけを入力順で出力する。

### ▼ 例

**入力**
```
5
apple pie
banana cake
pineapple tart
grape juice
apple juice
apple
```

**出力**
```
apple pie
pineapple tart
apple juice
```

### 制約
- 大文字小文字は区別してよい（そのまま部分一致）
- `contains()` を使ってもよい
