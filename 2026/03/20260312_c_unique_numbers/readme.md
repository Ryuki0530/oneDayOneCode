# 今日のC課題: Unique Numbers

## 問題
整数が `N` 個与えられます。  
**最初に登場した順番を保ったまま**、重複を除いて出力してください。

## 入力
1行目に整数 `N`  
2行目に整数 `N` 個

## 出力
重複を除いた整数列を、最初に出現した順に1行で空白区切りで出力してください。

## 制約
- `1 <= N <= 100`
- 各整数は `-1000 <= a_i <= 1000`

## 入力例1
```
5
1 2 1 3 2
```

## 出力例1
```
1 2 3
```

## 入力例2
```
8
4 4 4 4 4 4 4 4
```

## 出力例2
```
4
```

## ポイント
- CにはPythonのsetみたいな便利機能は標準でありません。
- すでに出た値かどうかを、配列を使って自分で調べます。
- 二重ループで十分解けます。
