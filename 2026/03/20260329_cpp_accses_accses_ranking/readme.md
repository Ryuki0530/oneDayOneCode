# 今日の課題（C++）

## 問題名
アクセス回数ランキング

## 問題
複数行のアクセスログが標準入力で与えられます。  
各行には `ユーザー名 ページ名` が空白区切りで書かれています。

各ページのアクセス回数を数え、以下の条件で出力してください。

- アクセス回数の多い順
- 回数が同じならページ名の辞書順

## 入力例
alice home
bob news
alice home
charlie news
alice profile
bob news
alice home

## 出力例
home 3
news 3
profile 1

## 条件
- 入力は EOF まで続く
- 1行に含まれるユーザー名・ページ名は英数字のみ
- 1 <= 行数 <= 100000

## ポイント
- `map` / `unordered_map`
- `vector`
- `sort`
- 比較関数

## 目標
集計して並べ替える処理に慣れること。

## 余裕があれば
ページごとのアクセス数だけでなく、  
「最も多くアクセスしたユーザー」も追加で出力してみてください。
