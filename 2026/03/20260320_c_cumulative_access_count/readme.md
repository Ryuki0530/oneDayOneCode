# 今日のC課題: 連続アクセス集計

## 問題
アクセスログが時刻順に与えられます。  
各ログは `時刻 ユーザーID` の形式です。

同じユーザーのアクセスが**直前のそのユーザーのアクセスから5秒以内**であれば、同じ連続アクセス中とみなします。  
**6秒以上空いたら新しい連続アクセス開始**とします。

各ユーザーについて、連続アクセスの回数を集計してください。

---

## 入力形式
1行目に整数 `N` が与えられます。  
続く `N` 行に、以下の形式でログが与えられます。

`t user_id`

- `t` は整数の時刻
- `user_id` は英数字からなる文字列
- ログは時刻昇順で与えられる

---

## 出力形式
ユーザーごとに以下の形式で出力してください。

`user_id 回数`

出力順は **user_id の辞書順昇順** とします。

---

## 条件
- `1 <= N <= 1000`
- `0 <= t <= 100000`
- `user_id` の長さは 1 以上 20 以下

---

## 例1

### 入力
```text
7
1 alice
3 bob
5 alice
12 alice
13 bob
20 bob
21 bob
```

### 出力
```text
alice 2
bob 2
```
