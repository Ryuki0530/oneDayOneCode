# TextUndoSimulator

## 概要

簡易テキスト編集シミュレータを作成してください。

文字列に対して、文字追加・削除・Undoを行い、最後の文字列を出力します。

## 入力

1行目に操作回数 `N` が与えられます。

続く `N` 行に、以下のいずれかの操作が与えられます。

```text
ADD c
DEL
UNDO
```

### 操作内容

- `ADD c`
  - 文字 `c` を文字列の末尾に追加する

- `DEL`
  - 文字列の末尾の1文字を削除する
  - 文字列が空の場合は何もしない

- `UNDO`
  - 直前に成功した `ADD` または `DEL` 操作を取り消す
  - 取り消せる操作がない場合は何もしない

## 出力

すべての操作後の文字列を1行で出力してください。

## 制約

- `1 <= N <= 100000`
- `c` は英小文字1文字

## 入力例

```text
8
ADD a
ADD b
DEL
ADD c
UNDO
UNDO
ADD d
ADD e
```

## 出力例

```text
abde
```

## 説明

操作の流れは以下の通りです。

```text
ADD a  -> a
ADD b  -> ab
DEL    -> a
ADD c  -> ac
UNDO   -> a
UNDO   -> ab
ADD d  -> abd
ADD e  -> abde
```

## ヒント

`string` と `stack` を使うと解きやすいです。

`ADD` や `DEL` が成功したときに、
「Undoされたら何をすれば元に戻るか」を履歴として保存すると実装しやすくなります。

例えば、

- `ADD a` が成功したら、Undo時には末尾を削除する
- `DEL` が成功したら、Undo時には削除した文字を末尾に戻す

という情報を保存します。
