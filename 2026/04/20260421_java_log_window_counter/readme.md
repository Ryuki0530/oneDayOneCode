# 今日の課題: LogWindowCounter

## 問題文
文字列で表されたアクセスログが N 件与えられます。
各ログは "時刻 ユーザー名" の形式です。

時刻は HH:MM 形式で、同じ日の 00:00〜23:59 の範囲とします。

各ユーザーについて、連続する10分間 に何回アクセスしたかを調べ、
そのユーザーの 10分間あたりの最大アクセス回数 を求めてください。

最後に、全ユーザー分を ユーザー名の昇順 で出力してください。

## 入力形式
1行目に整数 N が与えられる。
続く N 行に、以下の形式でログが与えられる。

時刻 ユーザー名

## 入力例
8
09:00 tanaka
09:03 tanaka
09:08 tanaka
09:20 tanaka
10:00 sato
10:04 sato
10:09 sato
10:30 sato

## 出力例
sato 3
tanaka 3

## 説明
- tanaka は 09:00, 09:03, 09:08 の3件が10分間以内に入るので最大は3
- sato は 10:00, 10:04, 10:09 の3件が10分間以内に入るので最大は3

## 条件
- 1 <= N <= 1000
- ユーザー名は英小文字のみ
- 入力は時刻順とは限らない

## 要求
- 各ユーザーごとに時刻をまとめる
- 時刻を分に変換して扱う
- ソートしてから、2ポインタ または スライディングウィンドウ で最大件数を求める

## 学習ポイント
- Map<String, List<Integer>>
- 時刻文字列の分変換
- Collections.sort
- 2ポインタ法

## 余裕があれば
同率最大だった時間帯の開始時刻も出力してください。
