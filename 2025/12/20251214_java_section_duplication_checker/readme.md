# Java課題：区間重複チェッカー（15分想定）

## 問題
複数の「時間区間」が与えられたとき、  
**1つでも重複している区間の組が存在するか**を判定するプログラムを作成せよ。

各区間は `[start, end]`（start <= end）で表される。

---

## 入力仕様
- 区間は `int[][] intervals` として与えられる
- 例：
```java
  int[][] intervals = {
      {1, 3},
      {5, 7},
      {2, 4}
  };
```
出力仕様
重複する区間が 1組でもあれば true

1つもなければ false を返す

例

```
{{1,3},{5,7},{2,4}}
```

出力
```
true
（[1,3] と [2,4] が重複）
```

```
{{1,2},{3,4},{5,6}}
```
出力
```
false
```
