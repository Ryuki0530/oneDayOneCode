# Python課題：ログファイルの集計（標準ライブラリのみ）

## 問題文
`access.log` という名前のテキストファイルがあり、  
各行には以下のような形式でアクセスログが記録されているとする。

```
2025-12-06 10:23:01 /index.html
2025-12-06 10:23:05 /login
2025-12-06 10:23:10 /index.html
2025-12-06 10:23:11 /profile
```

各行は  
`日時(YYYY-MM-DD HH:MM:SS)` と `アクセスされたパス`  
で構成されている。

---

## **課題**
Python を使って **最もアクセス数の多いパスとそのアクセス回数** を表示するスクリプト `main.py` を作成せよ。

### 要件
- 標準ライブラリのみ使用すること。
- `access.log` は同じディレクトリに存在するものとする。
- 出力例（例として `/index.html` が最多だった場合）：

```
Most accessed: /index.html (2 times)
```

---

## **テスト用 access.log の例**
以下の内容で `access.log` を作成して良い。

```
2025-12-06 10:23:01 /index.html
2025-12-06 10:23:05 /login
2025-12-06 10:23:10 /index.html
2025-12-06 10:23:11 /profile
2025-12-06 10:24:00 /index.html
```
