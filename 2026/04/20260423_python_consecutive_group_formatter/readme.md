# Today’s Problem: Consecutive Group Formatter

## 問題
1行の文字列が入力されます。  
この文字列について、**同じ文字が連続している部分ごと**に区切り、各グループを次の形式で出力してください。

`文字:連続回数`

## 入力
1行の文字列

## 出力
連続する文字ごとに、以下の形式で1行ずつ出力する。

`文字:連続回数`

## 入力例1
```text
aaabbccccdaa
```

## 出力例1
```text
a:3
b:2
c:4
d:1
a:2
```

## 入力例2
```text
z
```

## 出力例2
```text
z:1
```

## 入力例3
```text
abbaaa
```

## 出力例3
```text
a:1
b:2
a:3
```

## 条件
- 入力は空文字ではない
- 英小文字のみを含む
- 文字列の長さは 1 以上 1000 以下

## 要件
- `input()` を使って入力を受け取る
- 先頭から順に文字を見て処理する
- 結果を1行ずつ出力する

## ヒント
- 直前の文字と同じかどうかを見ながら進めるとよい
- 最後のグループの出力忘れに注意
