# 📡 リアルタイム・オーディオスペクトラムビューア（CLI 版）

あなたのタスクは、**PC で現在再生されている音**をリアルタイムに解析し、  
**周波数スペクトラムを CLI 上に ASCII アートで描画するツール**を作成することである。

使用言語は **Python** とする。

---

## 🎯 要件

### 1. オーディオ入力
- PC のスピーカー出力を “ループバック” で取得すること。
- Windows を前提とし、`soundcard` ライブラリの利用を許可する。
- 例：  
  `soundcard.default_speaker().recorder(loopback=True)` を利用してよい。

### 2. FFT による周波数解析
- 1 フレームごとに音声データを取得し、  
  `numpy.fft.rfft` を用いて **リアルタイム FFT** を行うこと。

### 3. バンド圧縮
- 高解像度スペクトラムそのままではなく、  
  **16 バンド程度の周波数帯に平均化**してよい。

### 4. CLI 描画
- 各バンドの強度を **横棒の ASCII グラフ**で表示すること：
  ```
  00 | ████
  01 | ███
  02 | ████████
  ...
  ```
- 画面クリアを行い、**毎フレーム更新表示**すること。

### 5. 実行条件
- 30〜60 FPS 程度の更新を目指す。
- ツールの終了は **Ctrl + C** で行うこと。
- 外部ライブラリの利用は以下を許可する：
  - `soundcard`
  - `numpy`

## 📚 注意点

- Windows では追加のステレオミキサー設定は不要とする。
- 実装は 1 ファイルのスクリプトでもよい。
- スペクトラムの正規化（スケーリング）は任意でよい。
