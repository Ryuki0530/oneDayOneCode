# Daily Coding Challenge (Python) — 「差分で管理するイベント集計」

## 問題
長さ `N` の数列 `A[0..N-1]`（初期値はすべて 0）がある。`Q` 個の操作を処理せよ。

操作は2種類：

- `ADD l r x` : 区間 `[l, r)`（0-index、rは含まない）に `x` を加算する
- `GET i` : `A[i]` の値を出力する

### 入力
```
N Q
(op_1)
(op_2)
...
(op_Q)
```

- `1 ≤ N ≤ 2×10^5`
- `1 ≤ Q ≤ 2×10^5`
- `0 ≤ l < r ≤ N`
- `|x| ≤ 10^9`
- `0 ≤ i < N`

### 出力
各 `GET` 操作ごとに1行、`A[i]` を出力。

## 制約/狙い
- `ADD` を愚直にやると O(NQ) で死ぬ。
- **差分配列（difference array）+ 前計算**で、全体を O(N+Q) で処理せよ。

## ヒント（超短）
`diff` を長さ `N+1` で持ち、
`ADD l r x` のとき `diff[l]+=x`, `diff[r]-=x`。
最後に prefix sum で `A` を復元し、`GET` は覚えておいて最後にまとめて答える。

## 例
入力:
```
5 6
ADD 1 4 3
GET 0
GET 2
ADD 0 2 -1
GET 1
GET 4
```

出力:
```
0
3
2
0
```
