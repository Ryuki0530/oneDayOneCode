# Problem: Sliding Window Maximum Sum

## 問題文
長さ N の整数配列 A と、整数 K が与えられる。  
連続する K 個の要素からなる部分配列の **合計値の最大** を求めよ。

## 入力形式
N K  
A1 A2 ... AN

## 出力形式
最大の合計値を 1 行で出力せよ。

## 制約
- 1 ≤ K ≤ N ≤ 2×10^5  
- -10^9 ≤ Ai ≤ 10^9  

## 例
### 入力
5 3  
1 2 -5 4 6  

### 出力
5

（部分配列 [ -5, 4, 6 ] の合計）

## 要求
- 計算量は **O(N)** で実装すること
- Java（標準入力・標準出力）で実装すること

## ヒント
- 最初の K 個の合計を作る
- 窓を 1 つ右にずらすごとに  
  「左端を引いて、右端を足す」
