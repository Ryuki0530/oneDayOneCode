# 今日のPython課題：差分配列で「区間加算」を高速処理

## 問題
長さ `N` の数列 `A`（初期値はすべて 0）がある。  
`Q` 個の操作が与えられるので、すべて処理した後の数列 `A` を出力せよ。

各操作は次の形式：
- `l r x`（1-indexed）
- 区間 `[l, r]` の各要素に `x` を加算する（`x` は負でもよい）

## 入力
- 1行目：`N Q`
- 続く `Q` 行：`l r x`

## 出力
最終的な `A1 A2 ... AN` を空白区切りで1行に出力。

## 制約
- `1 <= N, Q <= 2 * 10^5`
- `1 <= l <= r <= N`
- `-10^9 <= x <= 10^9`
- **O(NQ) は間に合わない**（差分配列で O(N+Q) を狙う）

## 例
入力：
```
5 3
1 3 10
2 5 -2
4 4 7
```

出力：
```
10 8 8 5 -2
```
