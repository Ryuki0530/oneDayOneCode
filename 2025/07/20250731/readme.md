# 📘 今日のC言語課題（2025/07/31）

## 📝 問題文：**文字列内のアルファベット頻度カウント**

標準入力から 1 行の文字列を受け取り、その中に含まれるアルファベット (`a`〜`z` / `A`〜`Z`) の出現回数を小文字に変換したうえでカウントし、`a`〜`z` の順に表示してください（出現しなかった文字も 0 を表示）。

### 🔍 入力

1 行の文字列（最大 1000 文字）

### 📤 出力
```
a: 出現数
b: 出現数
...
z: 出現数
```

### 🧪 入力例
```
Hello, World!
```


### ✅ 出力例
