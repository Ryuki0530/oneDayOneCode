# One Day One Code - Python課題
## 今日のテーマ
**ランキング付きアクセス集計**

## 問題
アクセスログが標準入力から複数行与えられます。  
各行は次の形式です。

`ユーザー名 操作名`

例:
`alice LOGIN`

入力の終わりまで読み込み、以下を出力してください。

1. 全ログ件数
2. ユーザーごとの出現回数
3. 操作ごとの出現回数
4. 最も多く出現したユーザー名
5. 最も多く出現した操作名

ただし、最多が複数ある場合は**辞書順で最も小さいもの**を採用してください。

---

## 入力例
```text
alice LOGIN
bob VIEW
alice VIEW
alice LOGOUT
bob VIEW
bob LOGOUT
```

## 出力例
```text
TOTAL 6
USERS
alice 3
bob 3
ACTIONS
LOGIN 1
LOGOUT 2
VIEW 3
TOP_USER alice 3
TOP_ACTION VIEW 3
```

---

## 出力ルール
- `TOTAL 件数`
- 次に `USERS`
- その後、ユーザー名の辞書順で `ユーザー名 回数`
- 次に `ACTIONS`
- その後、操作名の辞書順で `操作名 回数`
- 最後に
  - `TOP_USER ユーザー名 回数`
  - `TOP_ACTION 操作名 回数`

---

## 条件
- 1行につき要素は2つ
- 空行は入力されないものとする
- 1件以上のログが入力されるものとする

---

## ヒント
- 標準入力の複数行処理
- `dict`
- `sorted`
- `max` とキー関数
- `sys.stdin`

---

## 目標
Pythonでの基本的な
- 入力処理
- 連想配列による集計
- ソート
- 条件付きの最大値選択

を練習することが目的です。

---

## 余裕があれば
次の拡張にも挑戦してください。

### 拡張1
同じ形式で、ユーザーごとの「異なる操作種類数」も出力する。

### 拡張2
`TOP_USER` と `TOP_ACTION` を、同率1位が複数ある場合はすべて出力する。

### 拡張3
各操作について、その操作を行ったユーザー一覧も出力する。

## 関連モジュール
### `sys`
標準入力をまとめて処理するときによく使います。

### `collections`
`defaultdict` や `Counter` を使うと集計が楽になります。
