# 今日の課題（Java）: **Log Window Peak (Sliding Window Maximum)**

## 問題
長さ `N` の整数列 `A` と、ウィンドウ幅 `K` が与えられる。  
各 `i (0 <= i <= N-K)` について、区間 `A[i..i+K-1]` の **最大値** を順に出力せよ。

## 入力形式
- 1行目: `N K`
- 2行目: `A1 A2 ... AN`

## 出力形式
`N-K+1` 個の最大値を、空白区切りで1行に出力せよ。

## 制約
- `1 <= K <= N <= 2*10^5`
- `|Ai| <= 10^9`
- **O(N log N) だとギリ怪しい**ので、できれば **O(N)** を狙う。

## 例
入力
```
8 3
1 3 -1 -3 5 3 6 7
```

出力
```
3 3 5 5 6 7
```

## 要求
- Javaで実装すること
- 速い入出力を使うこと（`BufferedInputStream` 推奨）
- **単調キュー（deque）**で O(N) を達成すること

---

## ヒント（超短）
deque に「最大候補の添字」を入れる。  
- 後ろから `A[dequeLast] <= A[i]` を消す（弱い候補は追放）
- 範囲外（`i-K` より前）の先頭を消す
- `i>=K-1` になったら先頭が最大

---

## 仕上げ（任意）
同じ仕組みで「最小値」も出せるようにしてみよう（比較を逆にするだけ）。
