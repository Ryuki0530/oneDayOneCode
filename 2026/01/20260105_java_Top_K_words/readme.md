# 今日のJava課題：トップK頻出単語（安定ソート付き）

## 問題
標準入力で与えられる文章（複数行）から「単語」を抽出し、出現回数の多い順に **上位K個** を出力してください。

ただし、同じ出現回数の単語が複数ある場合は、**その単語が文章中で最初に出現した順**（安定）で並べてください。

---

## 入力形式
- 1行目：整数 `K`（1 ≤ K）
- 2行目以降：文章（0行以上、EOFまで）

---

## 単語の定義
- 英字 `A-Z a-z` の連続を1単語とする（それ以外の文字は区切り）
- 大文字小文字は区別しない（すべて小文字扱い）

例：  
`"Hello, hello!! Java_17"` → 単語列：`hello hello java`

---

## 出力形式
上位K個の単語を、次の形式で1行ずつ出力：

`<word> <count>`

※ 単語数がK未満しか存在しない場合は、存在する分だけ出力。

---

## 例
### 入力
```
3
Hello, hello!! Java_17 is nice.
Java is fast. HELLO?
```

### 出力
```
hello 3
java 2
is 2
```

（`java` と `is` は同数2回だが、文章中で `java` が先に初出）

---

## ねらい（学べること）
- 正規表現 or 文字走査でのトークナイズ
- `HashMap` で頻度カウント
- 「初出順」などのタイブレークを入れたソート
- `Collections.sort` / `stream` の使い分け

---

## 実装ヒント（推奨）
- `Pattern.compile("[A-Za-z]+")` で単語抽出
- `countMap: Map<String, Integer>`
- `firstIndexMap: Map<String, Integer>`（単語が最初に出た順番を連番で持つ）
- ソートキー：
  1) count 降順  
  2) firstIndex 昇順
