# 今日の課題: Bracket Reduction

## 問題概要
文字列が1つ与えられます。  
この文字列に対して、隣り合う `"()"` を見つけるたびに取り除く操作を、取り除けなくなるまで繰り返してください。  
最終的に残った文字列を出力してください。

## 入力
1行に文字列 `S`

## 制約
- `1 <= |S| <= 200000`
- `S` は `'('` と `')'` のみからなる

## 出力
操作を繰り返した後に残る文字列を出力してください。  
何も残らない場合は空行を出力してください。

## 入力例1
```text
(()())
```

## 出力例1
```text

```

## 入力例2
```text
())(()
```

## 出力例2
```text
)(
```

## 説明
例2では、以下のように消せます。

```text
())(()
-> )(()
-> )(
```

これ以上 `"()"` は存在しないので終了です。

## ヒント
- 実際に文字列を何度も走査して削除すると遅くなる可能性があります。
- 先頭から1文字ずつ見て、うまく「残っている部分」だけを管理すると高速にできます。

## 目標
- `O(N)` で解くことを目指してください。

## 追加課題
余裕があれば、`[]` や `{}` も含めて、
対応する括弧同士が打ち消し合う版に拡張してください。
```
