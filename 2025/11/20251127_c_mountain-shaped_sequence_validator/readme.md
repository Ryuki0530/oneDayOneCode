【問題文】

標準入力から N と N 個の整数列 a[0] … a[N-1] を読み込み、
その整数列が「山形」になっているか判定せよ。

ここで「山形」とは次の条件を満たすこととする：

  1. どこか 1 つのインデックス p が存在し、
  2. a[0] < a[1] < ... < a[p] まで単調増加、
  3. a[p] > a[p+1] > ... > a[N-1] まで単調減少。

p は 0 や N-1 ではなく、必ず 1 ～ N-2 のどこかである。

入力例:
7
1 3 5 8 6 4 2

出力例:
YES

別例:
5
1 2 3 4 5

出力例:
NO  (山の頂点がない)

【要件】

- 標準入力による読み取り
- 配列を使うこと
- p の探索は自由（先頭から・中央から・一度走査など自由）
- 出力は YES / NO のみ
