# 今日の課題（Java）

## 問題：ストリームで「移動平均との差分」を計算せよ

整数列 `A`（長さ N）と整数 `K` が与えられる。  
各 i (0 ≤ i < N) について、次を出力せよ。

- 区間 `[max(0, i-K+1), i]` に含まれる要素の **平均値（小数点以下切り捨て）** を `avg`
- `A[i] - avg` を出力

### 入力
```
N K
A1 A2 ... AN
```

### 出力
```
B1 B2 ... BN
```
### 制約
- 1 ≤ N ≤ 2×10^5
- 1 ≤ K ≤ N
- |Ai| ≤ 10^9

### 例
入力
```
5 3
1 2 3 4 5
```

出力
```
0 1 1 1 1
```


（例）
- i=0: 平均=1 → 1-1=0  
- i=2: 平均=(1+2+3)/3=2 → 3-2=1

### 要求
- **O(N)** で解くこと
- `long` を適切に使うこと
