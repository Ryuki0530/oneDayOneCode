# 今日の課題（Python・手短）

## テーマ  
**ログファイルから「連続する同一イベント」を検出する**

実務ログあるあるを、最小構成で切り取ります。  
アルゴリズムは軽め、入出力をちゃんと扱う系です。

---

## 問題文

次のようなテキストログファイル `events.log` が与えられる。
```
LOGIN
LOGIN
LOGIN
QUERY
QUERY
LOGOUT
LOGIN
LOGIN
```

### 要件

1. ログを **上から順に** 読み込む
2. **同じ行が3回以上連続した場合のみ** 検出対象とする
3. 検出結果を以下の形式で標準出力に表示せよ
```
LOGIN 3
```

（意味：`LOGIN` が 3 回連続）

---

## 制約

- Python **標準ライブラリのみ**
- ファイル名は `events.log` 固定
- 空行は無視してよい
- 出力順はログ出現順

---

## 実行例

```bash
python detect_repeat.py
```
出力：
```
LOGIN 3
```
