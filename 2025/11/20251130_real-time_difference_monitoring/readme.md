# 今日の C++ 課題：リアルタイム「差分監視」ミニツール

標準入力から逐次的に文字列が流れてくるとします。
あなたのプログラムは、前回の入力と比較し、「追加」「削除」「変更」された部分を画面に表示する差分ツールを作成してください。

-------------------------------
【 要件 】
-------------------------------

1. 標準入力から行単位で文字列を受け取る  
   （EOF または空行で終了してよい）

2. 前回の行との比較を行い、以下の形式で出力する：

   - 追加された部分 →  `+ <text>`
   - 削除された部分 →  `- <text>`
   - 変更されている場合 →  
         `- <old>`  
         `+ <new>`

3. 完全一致の場合は何も出力しない

4. C++ 標準ライブラリのみで実装する（正規表現・既存 diff ライブラリは禁止）

5. 比較は文字単位でよい


-------------------------------
【 入力例 】
-------------------------------

hello world  
hello worl  
hello wonderful world  


-------------------------------
【 出力例 】
-------------------------------

- d  
+ wonderful world  


-------------------------------
【 目的 】
-------------------------------

- 文字列処理の強化  
- 前回値を保持する状態管理  
- 小規模 diff アルゴリズムの自作練習  
- 観察力とデバッグ力の向上
