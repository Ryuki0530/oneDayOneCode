# **OneDayOneCode — C++ 課題（手短版）**

## 📘 問題：センサー値のノイズ除去（中央値フィルタ）

ある IoT デバイスから整数値のセンサーデータが流れてくる。  
ノイズの影響で急激に値が跳ねることがあり、ロボット制御（など重要な計算）の前に平滑化したい。

あなたの仕事は、**直近 N 個の値の中央値（median）を常に返すクラス `MedianFilter` を実装すること**である。

### 🔧 要件
- コンストラクタでウィンドウサイズ `N` を指定する。
- `add(int value)` を呼ぶたびに新しい値を追加し、  
  保持するのは常に「直近 N 個」だけ。
- `get_median()` で現在の中央値を返す。
- 値は整数でよい。
- ライブラリは C++ 標準まで（`<vector>`, `<algorithm>`, `<queue>` など自由）。

### 📝 入力例（テスト用の想定）
N = 5
入力: 3 1 9 2 7 8
中央値の出力:
3
1
3
2
3
7


### 🎯 目的
- データ構造（キュー or デック）、  
- 並べ替え、挿入、計算量の工夫など  
適当に工夫してもよし。

### 🚀 発展（任意）
- `O(logN)` で追加・中央値取得するために  
  「最大ヒープ＋最小ヒープ」の 2 ヒープ構成で作ってもよい。
