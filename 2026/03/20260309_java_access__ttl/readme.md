# 今日の1日1コード課題（Java）

## 問題名
TTLつきアクセス集計

## 問題
時刻付きのアクセス記録が順番に与えられます。  
各記録は `time user` の形式です。

ただし、**直近 W 秒以内**に存在する記録だけを有効とします。  
各時刻で、その時点の有効な記録に含まれる**異なる user の数**を出力してください。

## 入力
- 1行目: `N W`
- 続く N 行: `time user`

## 出力
各記録を読み込んだ直後の、有効範囲内にいる異なる user 数を1行ずつ出力してください。

## 制約
- `1 <= N <= 200000`
- `0 <= time <= 10^9`
- `1 <= W <= 10^9`
- 入力される `time` は昇順
- `user` は英数字文字列

## 例
### 入力
```text
7 5
1 alice
2 bob
4 alice
7 carol
8 bob
10 dave
11 alice
```

### 出力
```text
1
2
2
3
3
4
4
```
