# Daily Java Challenge: Duplicate Command Detector

## 目的

Javaで、入力されたコマンド履歴から「連続して同じコマンドが実行された回数」を集計するプログラムを作成してください。

## 問題文

N個のコマンド履歴が1行ずつ与えられます。

同じコマンドが連続して実行された場合、それを1つのグループとしてまとめ、以下の形式で出力してください。

コマンド名 回数

## 入力形式

N
command_1
command_2
...
command_N

## 制約

- 1 <= N <= 100
- コマンド名は英小文字のみ
- コマンド名の長さは1文字以上20文字以下

## 出力形式

連続した同じコマンドごとに、コマンド名と回数を出力してください。

## 入力例

8
open
open
save
save
save
close
open
open

## 出力例

open 2
save 3
close 1
open 2

## 実装条件

- Javaで実装すること
- 標準入力から読み込むこと
- Scanner または BufferedReader を使用してよい
- クラス名は Main とすること

## ヒント

直前のコマンドを保存しておき、現在のコマンドと同じならカウントを増やします。
違うコマンドが出たら、直前のコマンドとカウントを出力して、新しいグループを開始します。
