# 今日のJava課題: 文字出現ランキング

## 問題
1行の文字列を入力として受け取り、  
**各文字の出現回数を数えて、多い順に表示するプログラム**を作成してください。

## 仕様
- 入力は1行の文字列
- 空白も1文字として数える
- 大文字小文字は区別する
- 出現回数の多い順に表示する
- 出現回数が同じ場合は、**文字コードが小さい順**に表示する

## 出力形式
以下の形式で1文字ずつ出力してください。

`文字: 回数`

## 実行例1
### 入力
`banana`

### 出力
`a: 3`
`n: 2`
`b: 1`

## 実行例2
### 入力
`aabBC`

### 出力
`a: 2`
`B: 1`
`C: 1`
`b: 1`

## 条件
- `HashMap` または `TreeMap` の使用を推奨
- 並べ替えのために `List` 化して `sort` してよい

## 目標
- 文字ごとの出現回数を数える
- `Map` の基本操作に慣れる
- 条件付きソートを実装する

## 追加課題
余裕があれば、次にも挑戦してください。

1. 半角スペースを `[space]` と表示する
2. 入力が空文字なら `文字が入力されていません` と表示する
3. 英字だけを対象にして集計するモードを追加する

## ヒント
- `String` を `char` 単位で取り出すには `charAt`
- `Map<Character, Integer>` が使える
- `entrySet()` を `List` にしてカスタムソートするとよい

## 問題タイトル（ブランチ名用）
`character-frequency-ranking`
