# 今日の課題（C / 手短に終わる）

## 問題：ランレングス圧縮（RLE）の復元
ランレングス圧縮された文字列を復元して出力してください。

圧縮形式は以下：
- `文字` の後に **1以上の整数** が続く（繰り返し回数）
- 例：`A3B1C2` → `AAABCC`
- 文字は英大文字（`A`〜`Z`）のみ
- 入力は必ず正しい形式（数字は必ず1文字以上、回数は1以上）

## 入力
1行の文字列 `S`（長さは最大 10^5 程度を想定）

## 出力
復元後の文字列を1行で出力せよ。

※ 復元後は十分大きくなり得るが、今回のテストではメモリに載る範囲とする。

## 例
### 入力
A3B1C2

### 出力
AAABCC

## 追加要件（実装の注意）
- 数字が複数桁になるケースに対応すること（例：`Z12`）
- 文字列連結を雑にやると遅いので、`putchar`/`fwrite` などで逐次出力してOK（推奨）

## ヒント
- 文字を読んだら、次の数字列を `while ('0'<=S[i] && S[i]<='9')` で吸い取って回数 `k` を作る。
- `k` 回その文字を出力する。

---
### ミニテスト
入力: `X1Y2Z3` → 出力: `XYYZZZ`  
入力: `Q10` → 出力: `QQQQQQQQQQ`
