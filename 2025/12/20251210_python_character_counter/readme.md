# 🐍 Python ミニ課題（所要5〜10分）

## 問題  
標準入力から **1行の文章** を受け取り、その文章に含まれる  
**各文字の出現回数** を数えて、  
ASCII順（文字コード順）に並べて出力してください。

ただし、**空白（スペース）はカウント対象に含めない** ものとします。

### 入力例
```
Hello World
```

### 出力例
```
H:1
W:1
d:1
e:1
l:3
o:2
r:1
```

## 要件
- 使用ライブラリは **標準ライブラリのみ**。
- ASCII順ソートは `sorted()` を使用してよい。
- 空白以外はすべてそのままカウントする（大文字・小文字は区別する）。
- 出力形式は `文字:個数` を1行ずつ。
