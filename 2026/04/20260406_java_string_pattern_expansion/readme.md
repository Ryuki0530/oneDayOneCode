# 今日のJava課題

## 問題名
かっこ列の簡易展開

## 問題文
英小文字と数字と丸かっこからなる文字列 `S` が1行で与えられます。

この文字列では、次のような表現を使います。

- `3(ab)` は `ab` を 3 回繰り返すことを表す
- つまり `3(ab)` → `ababab`
- 入れ子もあり得る
- 例えば `2(a3(b))` → `abbbabbb`

文字列 `S` を展開した結果を出力してください。

## 入力例1
3(ab)

## 出力例1
ababab

## 入力例2
2(a3(b))

## 出力例2
abbbabbb

## 入力例3
x2(yz)3(a)

## 出力例3
xyzyzaaa

## 条件
- 入力は1行
- 数字は 1 桁の `1` ～ `9` のみとする
- 文字は英小文字と `(` `)` と数字のみ
- 展開後の文字列長は 10000 文字以下とする
- 入力は必ず正しい形式であるものとする

## ヒント
- 文字を左から順に読んで処理するとよいです
- `(` の前にある数字が「繰り返し回数」です
- 入れ子があるので、スタックを使うと整理しやすいです
- `StringBuilder` をうまく使うと効率よく実装できます

## 期待する学習ポイント
- `Stack` または `Deque` の利用
- `StringBuilder` の操作
- 文字列の逐次解析
- 入れ子構造の処理
