### 問題文

標準入力から空白区切りで与えられた整数列を読み取り、
**同じ数値が連続して出現した回数**をカウントしてください。

例えば、次のような入力があった場合：
```
    1 1 2 2 2 3 3 1 1 1 1
```
この場合の出力は：
```
    2 3 2 4
```
となります。各数字が「連続して何回出現したか」を表しています。

### 入力形式

空白区切りの整数列（1行）

### 出力形式

各連続する値の個数を、空白区切りで1行で出力してください。

### 制約

- 数列は1個以上1000個以下の整数（正または負の整数）からなる。
📌 実行例

入力
```
5 5 5 2 2 7
```
出力
```
3 2 1
```
