# Sliding Window Unique Counter

### 問題文
整数列 A と整数 K が与えられます。
長さ K の連続部分列ごとに、「その区間に含まれる異なる整数の個数」を順に出力してください。

### 入力
N K
A1 A2 ... AN

### 制約
1 <= K <= N <= 2*10^5
0 <= Ai <= 10^9

### 出力
長さ K の各連続部分列について、異なる整数の個数を1行ずつ出力してください。

### 入力例
```
7 3
1 2 1 3 2 3 3
```

### 出力例
```
2
3
3
2
```

### 説明
[1, 2, 1] -> 2種類
[2, 1, 3] -> 3種類
[1, 3, 2] -> 3種類
[3, 2, 3] -> 2種類
