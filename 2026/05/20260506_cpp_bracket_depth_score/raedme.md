# Problem: Bracket Depth Score

## 概要

文字列 `S` が与えられます。  
`S` は `(` と `)` のみで構成されています。

この文字列が正しい括弧列であるか判定し、正しい場合は最大のネスト深さを出力してください。  
正しくない場合は `INVALID` と出力してください。

## 入力

```text
S
```

## 出力

正しい括弧列なら最大のネスト深さを出力してください。  
正しくない括弧列なら `INVALID` と出力してください。

## 制約

- `1 <= |S| <= 100000`
- `S` は `(` と `)` のみからなる

## 入力例1

```text
(()(()))
```

## 出力例1

```text
3
```

## 入力例2

```text
())(
```

## 出力例2

```text
INVALID
```

## 実装方針

左から1文字ずつ見て、現在の深さ `depth` を管理する。

- `(` なら `depth++`
- `)` なら `depth--`
- 途中で `depth < 0` になったら不正
- 最後に `depth != 0` なら不正
- 途中で到達した最大の `depth` が答え

## 使用する主なC++要素

- `string`
- `for` 文
- `max`

## ブランチ名用タイトル

`bracket-depth-score`
