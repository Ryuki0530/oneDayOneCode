# 今日の課題: Consecutive Error Detector

## 問題
システムログを表す文字列が1行で与えられます。  
各文字は次の意味を持ちます。

- `N` : 正常
- `E` : エラー

このログの中に、**3回以上連続する `E`** が存在するかを判定してください。

## 入力
1行の文字列

### 例
`NNEEENEN`

## 出力
- 3回以上連続する `E` がある場合は `ALERT`
- ない場合は `OK`

## 入出力例

### 入力例1
`NNEEENEN`

### 出力例1
`ALERT`

---

### 入力例2
`NENENE`

### 出力例2
`OK`

## 条件
- 文字列長は 1 以上 1000 以下
- 文字列は `N` と `E` のみで構成される

## 要求
Javaで標準入力から文字列を受け取り、標準出力に結果を表示してください。

## 余裕があれば
- 連続した `E` の最大回数も表示する
- `E` が4回以上なら `CRITICAL` にする
