# 今日の課題（Python・手短）

## テーマ：ログの「連続重複」を圧縮する（RLE風だけど“連続だけ”）

サーバログやセンサ値など、**同じ文字列が連続して出る**ことがあります。  
これを「(回数, 文字列)」に圧縮して出力してください（連続している分だけまとめる）。

---

## 入力
- 1行目：整数 `N`（1 <= N <= 200000）
- 続く `N` 行：文字列 `s_i`（空白なし、長さ1〜200）

---

## 出力
連続する同一文字列をまとめ、各グループについて
```
count string
```


の形式で1行ずつ出力せよ。

---

## 例

### 入力
```
8
OK
OK
NG
NG
NG
OK
OK
OK
``` 

### 出力
```
2 OK
3 NG
3 OK
```


---

## 制約・注意
- `N` が大きいので **標準入力は `sys.stdin.buffer.readline` 推奨**
- 文字列の末尾改行は除去して扱うこと
- 連続していない同一文字列は別グループ（例：OK, NG, OK は 1 OK / 1 NG / 1 OK）

---

## 実装ヒント（超短）
- `prev` と `cnt` を持って1パスで処理
- 変化したら前のグループを出力、最後も忘れず出力

---

## 追加（余裕があれば）
出力を一度リストに貯めて `"\n".join(...)` でまとめて出す（高速化）
