# 🧩 今日のC++課題：最頻値（モード）を求める

## 問題
整数がスペース区切りで入力されます。  
その中で **最も多く出現する値（モード）** を求めてください。

- 出現回数が最大の値を1つ出力
- 複数ある場合は **最小の値** を出力

---

## 入力例
```
1 2 2 3 3 3 4 4
```

## 出力例
```
3
```

---

## 入力例2
```
5 1 1 2 2
```

## 出力例2
```
1
```

---

## 条件
- 入力は1行
- 要素数は最大10万個
- 値はint範囲

---

## ヒント
- `unordered_map` または `map` でカウント
- 「最大回数 + 最小値」の条件に注意

---

## 期待する実装
- 標準入力から読み取り
- モードを1つ出力

---

## 余裕があれば
- `map` と `unordered_map` の速度差を比較してみる
