# 今日のPython課題: Sequential Vote Counter

## 問題
投票ログを集計する簡単なプログラムを作成してください。

各行には `名前 半角スペース 値` の形式でデータが与えられます。  
`名前` は文字列、`値` は整数です。

同じ名前が複数回出ることがあります。  
各名前ごとの合計値を求めて、**合計値の降順**で出力してください。  
合計値が同じ場合は、**名前の辞書順昇順**で並べてください。

## 入力
標準入力から以下の形式で与えられます。

N  
name1 value1  
name2 value2  
...  
nameN valueN  

- `1 <= N <= 100`
- `名前` は英字のみからなる文字列
- `値` は `-1000` 以上 `1000` 以下の整数

## 出力
集計結果を以下の形式で出力してください。

name total

1行に1件ずつ出力してください。

## 入出力例

### 入力例1
5
alice 3
bob 10
alice 4
carol 8
bob -2

### 出力例1
alice 7
carol 8
bob 8

## 注意
出力順は「合計値の降順」なので、上の例では本来次の順になります。

carol 8  
bob 8  
alice 7

この条件を正しく満たすように実装してください。

## 追加課題
余裕があれば、最後に以下も出力してください。

- 登録された異なる名前の数
- 合計値が最も大きい名前

## ヒント
- 辞書 `dict` を使うと集計しやすいです。
- 並び替えには `sorted()` と `key` を使います。
- 降順と昇順が混ざるので、並び替え条件をよく考えてください。
