# 今日のPython課題：簡易ログフィルターツール

ログファイル（text）を読み込み、
指定したキーワードを含む行だけを抽出して表示するスクリプトを作成しなさい。

## 要件

- コマンドラインから以下の2つを受け取ること：
  1. ログファイル名
  2. 検索キーワード

- ログファイルを1行ずつ読み込み、検索キーワードを含む行だけ標準出力へ表示する。

- 検索キーワードは部分一致でよい。

- ファイルが存在しない場合はメッセージを表示して終了する。

## 入出力例（実行例）

```
python logfilter.py system.log ERROR
```

上記の場合、`system.log` から "ERROR" を含む行をすべて出力する。

## 注意
- Python 標準ライブラリのみを使用すること。
- Windows11 + VSCode 上で動く想定。
