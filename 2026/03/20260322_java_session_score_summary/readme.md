# 今日のJava課題

## 問題名
Session Score Summary

## 問題文
複数行のログが与えられます。
各行は、`ユーザー名 点数` の形式です。

同じユーザーが複数回登場することがあります。
各ユーザーについて、点数の合計を集計してください。

その後、以下の規則でユーザー一覧を出力してください。

- 合計点の降順
- 合計点が同じ場合はユーザー名の辞書順昇順

## 入力形式
標準入力から以下の形式で与えられます。

N
user1 score1
user2 score2
...
userN scoreN

- `1 <= N <= 1000`
- `score` は整数

## 出力形式
各ユーザーについて、次の形式で1行ずつ出力してください。

ユーザー名 合計点

## 入力例
6
alice 10
bob 7
alice 3
carol 15
bob 8
alice -2

## 出力例
bob 15
carol 15
alice 11

## 注意
並び順の条件をよく読んでください。

## ヒント
- `HashMap` で合計点を集計できます。
- 出力時には `List` に移してソートすると扱いやすいです。
- `Comparator` を使うと条件付きソートができます。

## 追加確認用入力


## 期待される出力
```
yamada 5

```
