# 今日のJava課題：最小の連続区間で目標和以上

## 問題
長さ `N` の正の整数配列 `A` と整数 `S` が与えられる。  
**連続部分配列の和が `S` 以上** となるもののうち、**長さが最小**のものを求め、その長さを出力せよ。  
そのような区間が存在しない場合は `0` を出力せよ。

## 入力形式
- 1行目：`N S`
- 2行目：`A1 A2 ... AN`

## 出力形式
- 条件を満たす最小長を1行に出力（存在しなければ `0`）

## 制約
- `1 <= N <= 200000`
- `1 <= Ai <= 10^9`（**正の数のみ**）
- `1 <= S <= 10^18`

## 例
入力
```
6 7
2 3 1 2 4 3
```

出力
```
2
```

説明：`[4,3]` の和が 7 で長さ 2 が最小。

## 要求
- `O(N)` を目指す（ヒント：**スライディングウィンドウ（尺取り法）**）
- `long` を使う（和が大きくなる）

## ヒント（最短で済む形）
- 右端 `r` を伸ばして `sum += A[r]`
- `sum >= S` の間、答えを更新しつつ左端 `l` を縮めて `sum -= A[l]`
