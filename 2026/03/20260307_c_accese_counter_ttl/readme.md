# 今日のC課題

## 問題名
TTLつきアクセス記録カウンタ

## 問題文
時刻付きのアクセス記録が順番に与えられます。  
各記録は「その時刻にそのユーザーが1回アクセスした」ことを表します。

ただし、**直近 `W` 秒以内**のアクセスだけを有効とみなします。

各クエリ時点で、有効なアクセス数を出力してください。

---

## 入力形式
1行目に2つの整数 `Q`, `W` が与えられる。

- `Q`: クエリ数
- `W`: 有効期間（秒）

続く `Q` 行は次のいずれか。

- `ADD t user`
  - 時刻 `t` にユーザー `user` のアクセスを追加する
- `COUNT t`
  - 時刻 `t` の時点で有効なアクセス数を出力する

---

## 有効なアクセスの定義
時刻 `t` において、アクセス時刻を `x` とすると、

`x > t - W`

を満たすアクセスだけが有効です。

つまり、**ちょうど `t-W` のものは無効**です。

---

## 制約
- `1 <= Q <= 200000`
- `1 <= W <= 10^9`
- `0 <= t <= 10^9`
- `user` は英数字からなる文字列
- 入力は時刻昇順で与えられる

---

## 出力形式
各 `COUNT` クエリに対して、有効なアクセス数を1行ずつ出力せよ。

---

## 入力例
```text
8 5
ADD 1 alice
ADD 2 bob
COUNT 2
ADD 6 alice
COUNT 6
ADD 7 bob
ADD 8 carol
COUNT 8
```

## 出力例
```text
2
2
3
```
