# C言語課題：最大値と最小値の差を求める

## 問題文

標準入力から整数の列を受け取り、その中の最大値と最小値の差を出力してください。

- 入力は1行にスペース区切りで最大100個の整数が与えられます。
- 各整数は `-10000` 以上 `10000` 以下とします。

## 入力形式

```
a1 a2 a3 ... an
```

（例）  
```
3 -5 7 0 2 -1
```

## 出力形式

```
最大値と最小値の差
```

（例）  
```
12
```

## 実行例

### 入力
```
3 -5 7 0 2 -1
```

### 出力
```
12
```

## 制約
- 入力される数値は1個以上100個以下。
- すべて整数値。
