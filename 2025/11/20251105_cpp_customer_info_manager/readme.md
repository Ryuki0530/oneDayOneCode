# 📘 C++課題：顧客情報の並べ替えと検索

## 概要
顧客情報（名前、年齢、購入金額）を構造体で管理し、  
複数の顧客データを入力後、**指定条件で並べ替えと検索**を行うプログラムを作成してください。

## 要件

### 1. 構造体の定義
以下のメンバを持つ構造体 `Customer` を定義すること。

- `std::string name;` — 顧客名  
- `int age;` — 年齢  
- `double total;` — 購入金額（円）

### 2. 入力
最初に顧客の人数 `N` を入力し、続いて N 人分の情報を入力します。  
各行は以下のように空白区切りで与えられます：

名前 年齢 購入金額

例：
```
4
Tanaka 35 8200.5
Suzuki 28 3500.0
Kato 42 12400.0
Miyamoto 28 7100.0
```

### 3. 並べ替え
次に、以下の選択肢を入力します：

1 → 購入金額の降順でソート
2 → 年齢の昇順でソート

入力に応じて、顧客リストをソートしてください。

### 4. 検索機能
次に、検索したい「年齢」を入力します。  
該当する年齢の顧客を一覧表示します。  
該当者がいない場合は `"該当者なし"` と出力してください。

### 5. 出力
- 並べ替え後の全顧客リストを表示。
- その後、検索結果を表示。

出力形式（例）：
```
=== Sorted List ===
Kato 42 12400.0
Tanaka 35 8200.5
Miyamoto 28 7100.0
Suzuki 28 3500.0

=== Search (age=28) ===
Miyamoto 28 7100.0
Suzuki 28 3500.0
```

## 実装ヒント
- `std::sort` とラムダ式を使うと簡潔に書けます。
- 検索には `std::vector` をループして条件分岐するだけでOKです。
- `iomanip` を使って小数点以下を整形すると見やすくなります。
