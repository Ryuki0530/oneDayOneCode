# Problem: Consecutive Duplicate Counter

## 概要
標準入力で複数行の文字列を受け取り、**直前の行と同じ内容が何回連続したか**を数えて出力してください。

時間がない前提なので、実装はシンプルでOKです。

## 入力
- 1行目に整数 `N`
- 続く `N` 行に文字列

## 出力
各行について、その行の文字列が**直前と同じなら連続回数を増やし**、違うなら `1` に戻して出力してください。

## 例

### 入力
~~~text
7
apple
apple
banana
banana
banana
apple
apple
~~~

### 出力
~~~text
1
2
1
2
3
1
2
~~~

## 説明
- 1行目の `apple` は最初なので `1`
- 次の `apple` は直前と同じなので `2`
- `banana` に変わったので `1`
- 次の `banana` は `2`
- 次の `banana` は `3`
- `apple` に変わったので `1`

## 条件
- `1 <= N <= 1000`

## 追加条件
- 文字列は大文字小文字を区別する
- 空文字列は出ないものとする

## 目標
- まずはそのまま実装
- 余裕があれば、出力を最後にまとめて行う形にもしてみる
