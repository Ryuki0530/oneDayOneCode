# 今日のC課題：最頻値を求めよう

## 問題文
N 個の整数が与えられます。  
最も多く出現した数（最頻値, mode）を求めて出力してください。  
もし最頻値が複数ある場合は、**最も小さい値**を出力してください。

---

### 【入力形式】
```
N
a1 a2 a3 ... aN
```
### 【出力形式】
```
mode
```
---

### 【入力例1】
```
7
3 1 2 3 2 3 1
```

### 【出力例1】
```
3
```
### 【入力例2】
```
5
4 4 2 2 1
```
### 【出力例2】
```
2
```

---

### 【制約】
- 1 ≤ N ≤ 100000  
- -10^6 ≤ ai ≤ 10^6  

---

### 【ヒント】
配列をソートしてからカウントする方法がシンプルで高速です。  
辞書構造が使えないCでは、ソート＋線形走査が定番手法です。
