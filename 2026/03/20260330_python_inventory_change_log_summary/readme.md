# 今日のPython課題: 在庫変動ログ集計

## 問題
商品の在庫変動ログが時系列順に与えられます。  
各ログは次の形式です。

`商品名 変動量`

- 変動量が正なら入荷
- 変動量が負なら出荷

すべてのログを処理したあと、各商品の最終在庫数を求めてください。

ただし、**最終在庫数が 1 以上の商品だけ**を出力してください。  
出力は**商品名の辞書順昇順**にしてください。

---

## 入力形式

1行目に整数 `N` が与えられる。  
続く `N` 行に、在庫変動ログが与えられる。

```text
N
item1 delta1
item2 delta2
...
itemN deltaN
```

---

## 制約

- `1 <= N <= 1000`
- 商品名は英小文字のみ、1文字以上20文字以下
- `-100 <= delta <= 100`

---

## 出力形式

最終在庫数が 1 以上の商品について、次の形式で1行ずつ出力してください。

```text
商品名 在庫数
```

商品名の辞書順で出力してください。

---

## 入力例1

```text
6
apple 10
orange 5
apple -3
banana 7
orange -5
apple -2
```

## 出力例1

```text
apple 5
banana 7
```

---

## 入力例2

```text
5
pen 3
pen -1
note 4
pen -2
note -4
```

## 出力例2

```text
(no output)
```

※ 在庫が1以上の商品がない場合は何も出力しない

---

## ポイント

- `dict` で商品ごとの在庫を管理する
- 存在しない商品は 0 個から開始する
- 最後に在庫が正のものだけ取り出してソートする

---

## 余裕があれば
次の拡張もやってみてください。

1. 在庫がマイナスになった商品もそのまま保持して確認できるようにする
2. 最終在庫が最も多い商品を最後に表示する
3. 入荷合計と出荷合計も別で集計する

---

## 関連モジュール・機能

### `dict`
キーごとに値を集計する基本。

### `sorted`
辞書順で並べるのに使う。

### `split`
1行の入力を商品名と変動量に分ける。
