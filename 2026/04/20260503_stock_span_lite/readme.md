# Daily C++ Challenge: Stock Span Lite

## 問題

N日分の株価が与えられます。

各日について、その日を含めて「直前から何日連続で、株価がその日の株価以下だったか」を求めてください。

## 入力

```text
N
P1 P2 ... PN
```

## 出力

各日の連続日数を空白区切りで出力してください。

## 制約

- 1 <= N <= 200000
- 1 <= Pi <= 1000000000

## 入力例

```text
7
100 80 60 70 60 75 85
```

## 出力例

```text
1 1 1 2 1 4 6
```

## 説明

- 1日目: 100 以下は自分だけ → 1
- 4日目: 70 以下は 60, 70 の2日連続 → 2
- 6日目: 75 以下は 60, 70, 60, 75 の4日連続 → 4
- 7日目: 85 以下は 80, 60, 70, 60, 75, 85 の6日連続 → 6

## 実装条件

- C++17で実装すること
- O(N^2) ではなく、O(N) を目指すこと
- stack または vector をスタックとして使うとよい

## ヒント

スタックに「株価」と「その株価がまとめて表す日数」を持たせる。

現在の株価以上になっている過去の日をまとめて取り除き、その日数を加算していく。
