# 今日のC課題：最短の「部分列」判定（Two Pointers）

## 問題
文字列 `S` と `T` が与えられる。`S` が `T` の**部分列（subsequence）**として含まれるなら `YES`、そうでなければ `NO` を出力せよ。

部分列とは：
`T` から0文字以上を削除して（順序は変えずに）`S` を作れるとき、`S` は `T` の部分列。

## 入力
- 1行目：文字列 `S`
- 2行目：文字列 `T`

（英小文字のみ、長さはそれぞれ最大 200000 程度を想定）

## 出力
- `S` が `T` の部分列なら `YES`
- そうでなければ `NO`

## 例
### 入力
```
abc
axbyc
```
### 出力
```
YES
```

### 入力
```
abc
acb
```
### 出力
```
NO
```

## 制約（実装上の狙い）
- O(|S|+|T|) で解くこと（全探索やDPは禁止気味）
- `scanf`/`fgets` などで高速に入力できるようにする
