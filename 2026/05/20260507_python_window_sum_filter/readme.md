# 今日の課題: Window Sum Filter

## 使用言語
Python

## 問題

整数列 `A` と整数 `K` が与えられます。

長さ `K` の連続区間ごとの合計を左から順に求め、その中で合計が `T` 以上になる区間の個数を出力してください。

## 入力

N K T
A1 A2 A3 ... AN

## 制約

- 1 <= K <= N <= 200000
- -1000 <= Ai <= 1000
- -10^9 <= T <= 10^9

## 出力

合計が `T` 以上になる長さ `K` の連続区間の個数を出力してください。

## 入力例

6 3 10
3 4 5 1 6 2

## 出力例

3

## 説明

長さ3の区間の合計は以下の通りです。

- 3 + 4 + 5 = 12
- 4 + 5 + 1 = 10
- 5 + 1 + 6 = 12
- 1 + 6 + 2 = 9

10以上の区間は3つなので、出力は `3` です。

## ヒント

毎回 `K` 個を足し直すと遅くなります。
最初の区間和を求めた後、左端を引いて右端を足す「しゃくとり」または「スライディングウィンドウ」で解きましょう。
