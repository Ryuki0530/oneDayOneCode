# 今日のPython課題: Log Summary CLI

## 問題
ログファイル `log.txt` を読み込み、各ユーザーごとのアクセス回数を集計して表示するプログラムを作成してください。

## ログ形式
`log.txt` の各行は次の形式です。

`ユーザー名,ステータスコード`

例:

```text
alice,200
bob,404
alice,200
charlie,500
bob,200
alice,404
```

## 要件
- `log.txt` を読み込む
- ユーザーごとの総アクセス回数を集計する
- さらに、ユーザーごとの `200` の回数も集計する
- 結果を以下の形式で表示する

```text
alice: total=3 success=2
bob: total=2 success=1
charlie: total=1 success=0
```

## 条件
- 表示順は、ユーザー名の昇順とする
- 辞書を使って集計する
- ステータスコードは文字列として扱ってよい

## 追加課題
余裕があれば、存在しないファイルの場合にエラーメッセージを表示してください。

## ヒント
- `with open(...) as f`
- `line.strip().split(",")`
- 辞書のネスト、または辞書を2個使う方法

## 目安
- 15〜25分程度
