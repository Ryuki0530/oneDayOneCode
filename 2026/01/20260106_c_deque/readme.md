# 今日のC課題：スライディングウィンドウ最大値（deque）

## 問題
長さ `N` の整数列 `A` と、窓幅 `K` が与えられる。  
各連続区間 `A[i..i+K-1]`（`0 <= i <= N-K`）の **最大値** を左から順に出力せよ。

ただし **全区間を毎回走査する O(NK) は禁止**。  
`deque（両端キュー）` を自作して **O(N)** で解け。

## 入力
- 1行目：`N K`
- 2行目：`A0 A1 ... A(N-1)`

## 出力
- `N-K+1` 個の最大値を空白区切りで1行に出力

## 制約
- `1 <= N <= 200000`
- `1 <= K <= N`
- `-10^9 <= Ai <= 10^9`

## 例
入力:
```
8 3
1 3 -1 -3 5 3 6 7
```

出力:
```
3 3 5 5 6 7
```
