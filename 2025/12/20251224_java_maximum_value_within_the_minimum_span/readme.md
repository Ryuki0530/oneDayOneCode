# 今日の課題（Java）: 「最小スパン内の最大値」ストリーミング

## 問題
整数列 `A`（長さ `N`）と整数 `K` が与えられる。  
連続部分列（長さ `K` の窓）ごとに **最大値** を求め、左から順に出力せよ。

ただし、`N` は最大で 200000 程度を想定し、全窓を愚直に走査すると間に合わない。  
**O(N)** を目標に実装せよ（ヒント：単調キュー / deque）。

## 入力形式
- 1行目: `N K`
- 2行目: `A1 A2 ... AN`

## 出力形式
長さ `K` の窓ごとの最大値を空白区切りで1行に出力せよ。  
（窓の個数は `N-K+1` 個）

## 制約（目安）
- `1 <= K <= N <= 200000`
- `-10^9 <= Ai <= 10^9`

## 例
### 入力
8 3
1 3 -1 -3 5 3 6 7

### 出力
3 3 5 5 6 7

## 実装要件
- `Deque<Integer>` を使い、**「候補の添字」**を管理すること
- deque 内の `A[添字]` は **単調減少** になるよう維持すること
- 窓の左端から外れた添字は先頭から捨てること
