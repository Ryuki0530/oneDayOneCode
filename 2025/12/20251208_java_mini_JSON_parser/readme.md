# Java 課題：ミニ JSON パーサ（キーと値の抽出）

次のような単純な JSON ファイルを読み取り、キーと値をすべて抽出して表示するプログラムを作成せよ。

入力ファイル `data.json`（UTF-8）:
```json
{
  "name": "Alice",
  "age": 20,
  "lang": "Java"
}
```
## 要件

1. JSON ライブラリ（Jackson, Gson など）は使用してはならない。
2. 対象とする JSON はトップレベルが `{}` のみであること。
3. 値は「文字列」または「数値」のみとし、ネスト構造は考えなくてよい。
4. 出力例:
```shell
name = Alice
age = 20
lang = Java
```
5. 余分なスペースや改行が含まれていても正しく処理できること。

## 目的

入出力処理と文字列処理を組み合わせて、実用的なデータ解析の基礎を身につける。
