# 今日のPython課題: 日本語文の階層構造抽出器

## テーマ
日本語の文章を入力し、単なる1列の文字列ではなく、
「どの語がどの語に係っているか」という階層構造として出力する簡易構文解析器を作成せよ。

## 課題内容
Python で、日本語文を入力として受け取り、
形態素解析と係り受け解析を行い、
その結果を木構造として表示するプログラムを作成せよ。

## 入力例
私は昨日学校へ行った
太郎は花子が読んでいた本を見つけた
彼は望遠鏡で鳥を見た

## 出力例
Sentence: 私は昨日学校へ行った

ROOT: 行った
├─ 私 は
├─ 昨日
└─ 学校 へ

Sentence: 太郎は花子が読んでいた本を見つけた

ROOT: 見つけた
├─ 太郎 は
└─ 本 を
    └─ 読んでいた
        └─ 花子 が

## 必須要件
- 日本語文を対象にすること
- 既存ライブラリを使ってもよい
- 解析結果を階層構造として表示すること
- パース不能または解析失敗時はその旨を表示すること

## 推奨
- GiNZA + spaCy を使う
- 余裕があれば JSON 形式でも保存する
- 余裕があれば複数文を連続処理できるようにする

## 発展
- 係り受けではなく、独自ルールで「節」の入れ子も推定する
- 「が」「を」「に」「と」などの助詞を使って簡易句構造を再構成する
- LLM向け前処理として、文章をネスト表現に変換する
