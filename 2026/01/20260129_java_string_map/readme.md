# 今日の課題（Java / 手短）

## 問題：最長連続区間（同じ種類がK種類以内）

長さ `N` の整数列 `A` が与えられる。  
**異なる値の種類数が高々 `K`** となるような **連続部分列（区間）** のうち、**最大の長さ**を求めよ。

- 例：`A = [1,2,1,3,4,2,3], K=2`  
  `1,2,1` は種類 {1,2} でOK（長さ3）  
  `2,1,3` は種類 {1,2,3} でNG（種類3）

## 入力
- 1行目：`N K`
- 2行目：`A1 A2 ... AN`

## 出力
- 条件を満たす連続部分列の最大長（整数）

## 制約（目安）
- `1 <= N <= 200000`
- `0 <= K <= N`
- `Ai` は 32bit int 範囲

## 要求
- `O(N)` で解くこと（**しゃくとり法 / スライディングウィンドウ**推奨）
- Javaで実装（`HashMap<Integer, Integer>` などで出現回数管理）

## ヒント（超要点だけ）
- 右端 `r` を伸ばしながら数をカウント
- 種類数が `K` を超えたら左端 `l` を縮めて調整
- 各時点で `r-l+1` の最大を更新

## サンプル
入力:
```
7 2
1 2 1 3 4 2 3
```

出力:
```
3
```
