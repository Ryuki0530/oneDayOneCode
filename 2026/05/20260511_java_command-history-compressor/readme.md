# コマンド履歴圧縮

## 問題文

ある開発用CLIツールでは、ユーザーが入力したコマンド履歴を保存している。

ただし、同じコマンドが連続して入力された場合は、履歴を圧縮して表示したい。

N 個のコマンドが順番に与えられるので、連続して同じコマンドが続いた区間ごとに、

- コマンド名
- 連続回数

を出力しなさい。

## 入力

```text
N
command_1
command_2
...
command_N
```

## 制約

- 1 <= N <= 100000
- command_i は英小文字、数字、`-`, `_` からなる文字列
- command_i の長さは 1 以上 50 以下

## 出力

連続して同じコマンドが続いた区間ごとに、以下の形式で出力しなさい。

```text
command count
```

## 入力例

```text
8
build
build
test
test
test
deploy
build
build
```

## 出力例

```text
build 2
test 3
deploy 1
build 2
```

## 補足

同じ `build` でも、途中に別のコマンドが挟まっている場合は別の区間として扱う。

## 実装条件

- Javaで実装すること
- クラス名は `Main` とすること
- 標準入力から読み込み、標準出力へ出力すること
