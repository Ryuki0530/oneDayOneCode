# 🐍 Python基礎問題：リスト内の偶数の合計

## 問題文

標準入力から複数の整数を受け取り、それらの中から偶数のみを抽出し、その合計を出力するプログラムを作成してください。

## 入力

- 1行にスペース区切りで整数が複数与えられます。
```
12 3 5 8 7 10
```


## 出力

- 偶数のみの合計を出力してください。
```
30
```


## 制約

- 入力される整数は -1000 以上 1000 以下とします。
- 整数は1個以上与えられます。

## 実行例

### 入力
```
4 9 2 7 6
```

### 出力
```
12
```
