# 今日の課題: Error Code Tally

## 問題
複数行のエラーコードログが与えられます。  
各行には `ユーザー名 エラーコード` の形式でデータが入っています。

同じエラーコードが何回出現したかを集計し、**出現回数の多い順**に表示してください。  
出現回数が同じ場合は、**エラーコードの文字列昇順**で表示してください。

## 入力形式
1行目に整数 `N` が与えられます。  
続く `N` 行に、以下の形式の文字列が与えられます。

`ユーザー名 エラーコード`

## 制約
- `1 <= N <= 1000`
- ユーザー名、エラーコードは空白を含まない文字列
- エラーコードは英大文字・数字・記号を含んでよい

## 出力形式
各エラーコードについて、以下の形式で1行ずつ出力してください。

`エラーコード 出現回数`

## 入力例1
```text
6
alice E001
bob E404
carol E001
dave E500
erin E404
frank E001
```

## 出力例1
```text
E001 3
E404 2
E500 1
```

## 入力例2
```text
5
u1 X9
u2 A1
u3 X9
u4 A1
u5 B2
```

## 出力例2
```text
A1 2
X9 2
B2 1
```

## ポイント
- `HashMap` で回数を数える
- `Map.Entry` のリスト化
- `sort` で複数条件ソートを行う

## 追加条件
- 入力は標準入力から受け取ること
- クラス名は `Main` とすること

## 学習テーマ
- `HashMap`
- `ArrayList`
- `Collections.sort` または `List.sort`
- Comparator を使った並び替え
