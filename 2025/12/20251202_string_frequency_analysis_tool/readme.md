## 📝 課題：文字列の頻度解析ツールを作成せよ

標準入力から複数行のテキストを読み込み、  
各単語が出現した回数をカウントして出力するプログラムを作成せよ。

### 【仕様】
1. 標準入力から EOF（Ctrl+D / Ctrl+Z）までテキストを受け取る。
2. 単語は空白・句読点で区切るものとする。
3. 出現回数の多い順に並べて出力する。
4. 形式は`単語: 回数`で表示する。
5. 大文字小文字は区別しない（全部小文字として集計）。
6. 英数字以外の文字は単語分割として扱う。

### 【例】
入力：
```
Hello, world!
hello Java world.
```

出力：
```
hello: 2
world: 2
java: 1
```
