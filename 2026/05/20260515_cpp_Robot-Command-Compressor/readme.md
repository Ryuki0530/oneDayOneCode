# 今日の課題: Robot Command Compressor

## 概要

ロボットに送る移動命令列を、できるだけ短い形に圧縮するプログラムを作成してください。

ロボットには以下の4種類の命令があります。

- `U` : 上へ1マス移動
- `D` : 下へ1マス移動
- `L` : 左へ1マス移動
- `R` : 右へ1マス移動

ただし、連続する同じ命令は以下の形式に圧縮できます。

```text
命令文字 回数
```

例えば、次の命令列は

```text
UUUDDLLLLR
```

以下のように圧縮できます。

```text
U 3
D 2
L 4
R 1
```

また、最終的なロボットの座標も求めてください。

初期位置は `(0, 0)` とします。

- `U` は y を +1
- `D` は y を -1
- `R` は x を +1
- `L` は x を -1

## 入力

```text
S
```

## 制約

- `S` は `U`, `D`, `L`, `R` のみからなる文字列
- `1 <= |S| <= 100000`

## 出力

まず、圧縮後の命令数 `K` を出力してください。

続けて、圧縮後の命令を `K` 行出力してください。

最後に、最終座標 `x y` を出力してください。

## 入力例1

```text
UUUDDLLLLR
```

## 出力例1

```text
4
U 3
D 2
L 4
R 1
-3 1
```

## 入力例2

```text
RURURUDD
```

## 出力例2

```text
8
R 1
U 1
R 1
U 1
R 1
U 1
D 2
3 1
```

## 実装条件

- C++で実装すること
- `std::string` を使うこと
- 文字列を左から1回走査して圧縮すること
- 計算量は `O(N)` を目標にすること

## 余裕があれば追加実装

以下の情報も最後に追加で出力できるようにしてください。

```text
DIST d
```

ここで `d` は、最終座標から原点までのマンハッタン距離です。

マンハッタン距離は次の式で求められます。

```text
abs(x) + abs(y)
```
