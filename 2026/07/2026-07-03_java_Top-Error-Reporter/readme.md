# OneDayOneCode

## タイトル
Java - Top Error Reporter

## 問題

アプリケーションのエラーログ一覧から、
エラーコードごとの発生回数を集計し、
発生回数の多い順に表示してください。

入力データ

```java
List<String> logs = List.of(
    "E101",
    "E404",
    "E101",
    "E500",
    "E404",
    "E404",
    "E101",
    "E301",
    "E500",
    "E404"
);
```

## 出力例

```
E404 : 4
E101 : 3
E500 : 2
E301 : 1
```

## 条件

- HashMap を利用して集計すること
- Comparator またはラムダ式で件数の多い順にソートすること
- 同じ件数の場合はエラーコードの昇順とすること
- メソッドを最低1つ作成すること

## 学習ポイント

- HashMap
- getOrDefault()
- entrySet()
- Comparator.comparing()
- ラムダ式
- Listへの変換とソート
