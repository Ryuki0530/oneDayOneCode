# 今日のPython課題：ISSトラッカー（5分おき更新）

## 概要
ISS（国際宇宙ステーション）の現在位置を取得できる公開APIを使い、**5分に1回の更新**で「いまISSが地球上のどこを飛んでいるか」を表示するミニアプリを作成せよ。

- 目的：API利用 / 定期更新 / 表示（可視化 or 整形出力）の基本を一気に体験する
- 想定環境：Windows 11 + VSCode + Python 3.x

---

## 使用するAPI
以下のどちらか **片方** を使う（おすすめは A）。

### A. wheretheiss.at（おすすめ）
- エンドポイント：`https://api.wheretheiss.at/v1/satellites/25544`
- 返ってくる主な値（例）：`latitude`, `longitude`, `altitude`, `velocity`, `timestamp`

### B. open-notify
- エンドポイント：`http://api.open-notify.org/iss-now.json`
- 返ってくる主な値（例）：`iss_position.latitude`, `iss_position.longitude`, `timestamp`

---

## 要件（最低ライン）
1. **5分に1回**、ISSの位置情報をAPIから取得する  
2. 取得したデータから少なくとも以下を表示する  
   - 緯度（latitude）
   - 経度（longitude）
   - 取得時刻（timestamp）
3. ネットワークエラー時に落ちず、**エラーメッセージを表示して次回更新を継続**する

---

## 表示方法（どれか1つ選ぶ）
### 方式1：CLI表示（最短）
- 例）更新のたびに最新位置を表示  
- できれば「前回からどれだけ移動したか（距離）」も表示すると良い

### 方式2：簡易地図表示（おすすめで面白い）
- ローカルでHTMLを出力し、地図上にピンを表示
- 更新のたびにピン位置を動かす（ページ自動更新でもOK）

※地図はOpenStreetMap + Leafletなどを使う想定でよい（Python側はHTML生成に徹する形でもOK）

---

## 追加要件（余力があれば）
- 過去N回分（例：12回＝1時間）の位置を保存し、**軌跡（線）**を表示する
- 速度（km/h）や高度（km）も表示する（APIが返す場合）
- 前回地点からの移動距離（km）を計算して表示する（ハバースイン距離など）

---

## 実装ルール
- APIキー不要の範囲で実装する（今回は不要）
- 更新間隔は定数で管理する（例：`UPDATE_SECONDS = 300`）
- ファイル構成は自由だが、最低でも `main.py` を用意すること

---

## 入力/出力仕様（CLI方式の場合）
### 入力
なし（起動したら自動で更新開始）

### 出力（例）
[2026-01-16 21:00:00] lat=35.1234, lon=139.1234
[2026-01-16 21:05:00] lat=36.0001, lon=140.0002 (+98.2 km)


---

## ヒント（使ってよい）
- HTTP取得：`requests` もしくは標準ライブラリ `urllib.request`
- JSONパース：`json`
- 定期実行：`time.sleep()` / `sched` / `asyncio` どれでも可
- ログ保存：`csv` / `jsonl`（1行1レコード）がおすすめ
