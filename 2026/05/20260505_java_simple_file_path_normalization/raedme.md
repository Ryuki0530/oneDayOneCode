# 今日の課題：簡易ファイルパス正規化

## 概要

UNIX風のファイルパスが与えられます。

`.` は「現在のディレクトリ」、
`..` は「1つ上のディレクトリ」を表します。

与えられたパスを正規化して、不要な `.` や `..` を取り除いた絶対パスを出力してください。

## 入力

1行で文字列 `S` が与えられます。

```text
S
```

## 条件

- `S` は `/` から始まる絶対パス
- ディレクトリ名は英小文字のみ
- `/` が連続することがある
- `.` や `..` が含まれることがある
- 文字列長は 1 以上 200 以下

## 出力

正規化後の絶対パスを1行で出力してください。

## 入力例1

```text
/home/user/./docs/../image
```

## 出力例1

```text
/home/user/image
```

## 入力例2

```text
/a//b/../../c/
```

## 出力例2

```text
/c
```

## 入力例3

```text
/../a/b
```

## 出力例3

```text
/a/b
```

## 補足

ルートディレクトリ `/` より上には移動できません。

例えば `/../a` は `/a` になります。

## 実装のヒント

- `/` で文字列を分割する
- 空文字列と `.` は無視する
- `..` が来たら、現在のディレクトリリストから末尾を削除する
- 通常のディレクトリ名ならリストに追加する
- 最後に `/` で結合する

## Javaで使うとよさそうなもの

```java
String[] parts = path.split("/");
ArrayList<String> dirs = new ArrayList<>();
String.join("/", dirs);
```
