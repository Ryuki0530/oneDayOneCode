# 📞 Python Mini Challenge — Phone Number Inspector

本課題では、Python の `phonenumbers` ライブラリを用いて  
**電話番号の解析・正規化・地域情報の取得を行う CLI アプリケーション** を作成せよ。

---

## 📝 課題内容

ユーザーが任意の文字列を入力すると、その文字列に含まれる電話番号を自動検出し、  
以下の情報を一覧で表示するプログラムを作成せよ。

1. **正規化された電話番号 (E.164 形式)**  
2. **国情報（国名）**  
3. **キャリア情報（取得できる場合）**  
4. **地理情報（地域名）**  
5. **有効な電話番号かどうかの判定**

入力文字列には複数の電話番号が含まれる場合がある。

---

## 🎯 要件

- Python で `phonenumbers` を利用すること。
- 入力は1行の文字列。例:  
  `"Please call me at +81 90-1234-5678 or 03-1234-5678"`
- 上記文字列から電話番号を **すべて抽出** すること。
- 各電話番号について、以下を出力せよ:
```
[Raw] 03-1234-5678
[E164] +81312345678
[Region] Japan
[Carrier] N/A
[Valid] True
```
---

## 🚀 実行例

入力:
```
Contact: +1 415-555-2671 / 090-1234-5678
```

出力:
```
[Raw] +1 415-555-2671
[E164] +14155552671
[Region] United States
[Carrier] Verizon (maybe)
[Valid] True

[Raw] 090-1234-5678
[E164] +819012345678
[Region] Japan
[Carrier] NTT Docomo (maybe)
[Valid] True
```


---

## 📦 事前準備

```
pip install phonenumbers
```


---

## 🎉 目標

完成すると、ユーザーが適当な文章を入力しただけで  
**中に埋もれている電話番号をすべて解析してくれる電話番号インスペクタ**が出来上がる。

電話番号という世界の裏側を覗く良い練習になる。
