# 今日のPython課題

## 問題名
イベント集計ミニログ解析

## 問題
標準入力から複数行のログを読み取り、ユーザーごとの出現回数を集計してください。

各行は次の形式です。

`ユーザー名 アクション名`

例えば:

`alice login`

入力の終わりまで読み込み、以下を出力してください。

1. 全ログ件数
2. ユーザーごとの件数を、ユーザー名の辞書順で出力

## 入力例
alice login
bob view
alice view
alice logout
bob logout

## 出力例
TOTAL 5
alice 3
bob 2

## 条件
- 1行につき2要素あるとする
- 空行は無視してよい
- 標準入力の終わりまで処理すること

## ヒント
- `sys.stdin`
- `dict` または `collections.defaultdict(int)`

## 期待する流れ
- 1行ずつ読む
- 総件数を数える
- ユーザーごとの件数を増やす
- 最後に辞書順で出力する

## おまけ
余裕があれば、アクションごとの件数も追加で集計してみてください。
