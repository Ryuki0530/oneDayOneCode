# 今日の課題: Rest Timer

## 概要

体調管理用の簡易タイマー判定プログラムを作成してください。

N個の作業時間が分単位で与えられます。
それぞれの作業時間について、以下のルールで出力してください。

- 30分未満: `OK`
- 30分以上60分未満: `REST`
- 60分以上: `STOP`

## 入力

1行目に整数 N が与えられる。
2行目に N 個の整数 t_i が空白区切りで与えられる。

## 出力

各作業時間に対する判定を1行ずつ出力してください。

## 制約

- 1 <= N <= 100
- 0 <= t_i <= 300

## 入力例

5
10 30 45 60 120

## 出力例

OK
REST
REST
STOP
STOP

## 実装条件

- Pythonで実装すること
- 標準入力から読み込むこと
- 標準出力に出力すること
