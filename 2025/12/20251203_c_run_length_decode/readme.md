# C Programming Task — Run-Length Decode（可変長の復号）

圧縮表現された文字列を **伸張（decode）** するプログラムを作成せよ。

## 入力形式
1 行の文字列が標準入力から与えられる。
形式は以下のように、  
**「アルファベット + 正の整数」** の組が連続しているとする。

例：
A3B1C5

これは  
`A` が 3 回  
`B` が 1 回  
`C` が 5 回  
出現することを意味する。

## 出力形式
伸張後の文字列を出力せよ。

例：
入力：
A3B1C5

出力：
AAABCCCCC


## 制約
- アルファベットは英字大文字のみ  
- 繰り返し回数は 1〜999 の整数  
- 不正入力のチェックは不要
