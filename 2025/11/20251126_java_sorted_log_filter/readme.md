# Java 課題：ソート付きログフィルタ

## 概要
標準入力から複数行のログ文字列を読み込み、
その中から「ERROR」を含む行のみを抽出し、
タイムスタンプ順にソートして出力してください。

## 入力形式
- 標準入力から複数行のテキストが与えられる。
- 終了は EOF とする。（VSCode のターミナルなら Ctrl+D）

### ログ行の形式（例）
2025-11-26 18:31:10 INFO User logged in
2025-11-26 18:32:08 ERROR Failed to connect
2025-11-26 18:33:41 WARN Low memory
2025-11-26 18:33:55 ERROR Disk full

## 要件
1. 行内に "ERROR" を含むものだけ抽出する。
2. 抽出した行を、先頭のタイムスタンプで昇順ソートする。
3. ソート後の行をすべて出力する。
4. タイムスタンプのパースには `LocalDateTime` を用いること。

## 出力例（上記入力の場合）
2025-11-26 18:32:08 ERROR Failed to connect
2025-11-26 18:33:55 ERROR Disk full
